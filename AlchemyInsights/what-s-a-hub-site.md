# <a name="whats-a-hub-site"></a>Co to jest witryna centrum?

Witryna centrum pomaga organizować witryny i nadać im wspólną nawigację i znakowanie. Administratorzy globalni lub program SharePoint będą mogli tworzyć witryny centrum i określać, którzy użytkownicy mogą kojarzyć swoje witryny z lokacją centrum. Aby uzyskać więcej informacji, zobacz [Witryny centrum programu SharePoint](https://go.microsoft.com/fwlink/?linkid=869388).
