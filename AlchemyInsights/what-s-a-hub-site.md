# <a name="whats-a-hub-site"></a>Co to jest witryna koncentratora?

Lokacji centralnej ułatwia organizowanie witryn i nadać im wspólna nawigacja i znakowania. Szablon globalny lub program SharePoint administratorzy będą mogli tworzyć witryny koncentratora i określić, którzy użytkownicy można skojarzyć ich witryn z lokalizacji centralnej. Aby uzyskać więcej informacji zobacz [witryny programu SharePoint koncentrator nowości w usłudze Office 365](https://go.microsoft.com/fwlink/?linkid=869388).
