# <a name="whats-a-hub-site"></a>Co to jest witryna centrum?

Witryna centrum pomaga organizować witryny i dawać im wspólną nawigację i markę. Administratorzy globalni i Administratorzy programu SharePoint będą mogli tworzyć witryny centrum i określać użytkowników, którzy będą mogli kojarzyć witryny z witryną centrum. Aby uzyskać więcej informacji, zobacz [witryny Centrum programu SharePoint](https://go.microsoft.com/fwlink/?linkid=869388).
