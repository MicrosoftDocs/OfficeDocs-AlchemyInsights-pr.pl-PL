# <a name="microsoft-365-f3-subscription-and-license-management-faq"></a>Subskrypcja Microsoft 365 F3 i zarządzanie licencjami — często zadawane pytania

- Aby **poznać i kupić** subskrypcję usługi F3, zobacz [Microsoft 365 F3](https://www.microsoft.com/microsoft-365/microsoft-365-enterprise-f3?activetab=pivot%3aoverviewtab).
- Aby **przełączyć plany**, zobacz [Ręczne przełączanie planów platformy Microsoft 365 dla firm](https://docs.microsoft.com/microsoft-365/commerce/subscriptions/switch-plans-manually?view=o365-worldwide).
- Aby **przypisać** użytkownikom licencje usługi F3, przejdź do pozycji **Rozliczenia > [Licencje](https://go.microsoft.com/fwlink/p/?linkid=842264)**. Aby uzyskać więcej informacji, zobacz [Przypisywanie licencji](https://docs.microsoft.com/microsoft-365/admin/manage/assign-licenses-to-users?view=o365-worldwide).
- Aby **dodać/usunąć** licencje usługi F3, przejdź do pozycji **Rozliczenia > [Produkty i usługi](https://go.microsoft.com/fwlink/p/?linkid=842054)** i kliknij pozycję **Dodaj/Usuń licencje**. Aby uzyskać więcej informacji, zobacz [Dodawanie/usuwanie licencji](https://docs.microsoft.com/microsoft-365/commerce/licenses/buy-licenses?view=o365-worldwide#add-or-remove-licenses-for-your-business-subscription). 
- Aby **anulować ** licencje usługi F3, przejdź do pozycji **Rozliczenia > [Produkty i usługi](https://go.microsoft.com/fwlink/p/?linkid=842054)**, kliknij pozycję **Ustawienia i akcje**, a następnie kliknij pozycję **Anuluj subskrypcję**. Aby dowiedzieć się więcej, zobacz [Anulowanie subskrypcji](https://docs.microsoft.com/officF365/admin/subscriptions-and-billing/cancel-your-subscription).
- Aby **aktywować** subskrypcję usługi F3, zobacz [Aktywowanie subskrypcji](https://docs.microsoft.com/alchemyinsights/activate-your-office-365-subscription).
- Aby **ponownie aktywować** subskrypcję usługi F3, zobacz [Ponowne aktywowanie subskrypcji](https://docs.microsoft.com/alchemyinsights/reactivate-your-subscription).
- Aby **przedłużyć** okres próbny usługi F3, zobacz [Przedłużanie okresu próbnego](https://docs.microsoft.com/alchemyinsights/extend-your-trial-for-office-365-for-business).

Aby uzyskać więcej informacji, zobacz [Zarządzanie licencjami subskrypcji](https://docs.microsoft.com/microsoft-365/commerce/licenses/buy-licenses?view=o365-worldwide#add-or-remove-licenses-for-your-business-subscription).
