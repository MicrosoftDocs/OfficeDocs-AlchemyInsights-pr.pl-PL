# <a name="issues-seeing-calendar-freebusy-information"></a>Problemy z wyświetlaniem informacji o kalendarzu wolny/zajęty

Jeśli w innych kalendarzach programu Outlook nie widać informacji o kalendarzu wolny/zajęty lub jeśli inne osoby nie widzą informacji wolnych/zajętych w kalendarzu, użyj Asystenta pomocy technicznej i odzyskiwania firmy [Microsoft dla usługi Microsoft 365](https://diagnostics.office.com/).
