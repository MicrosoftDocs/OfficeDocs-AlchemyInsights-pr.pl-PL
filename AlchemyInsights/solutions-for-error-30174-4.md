# <a name="error-code-30174-4-when-installing-office"></a>Kod błędu 30174-4, podczas instalowania pakietu Office

Ten błąd może wystąpić, jeśli program antywirusowy, zapora, ustawienia serwera proxy lub połączenie uniemożliwiają instalację pakietu Office. Oto kilka rzeczy, możesz spróbować rozwiązać problem.
  
- Używając połączenia przewodowego

- Tymczasowo wyłącz ustawienia serwera proxy

- Tymczasowo wyłącz oprogramowanie antywirusowe
