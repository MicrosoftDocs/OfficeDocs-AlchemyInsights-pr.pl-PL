# <a name="turn-an-add-in-off-for-outlook-for-windows"></a>Włączanie dodatku dla programu Outlook dla systemu Windows

Jeśli nie chcesz używać dodatku w programie Outlook 2013 lub Outlook 2016 dla systemu Windows, możesz wyłączyć tę funkcję, aby nie była wyświetlana w wiadomościach.  

1. W programie Outlook kliknij pozycję **plik > Zarządzanie dodatkami**.
2. Kliknij pozycję  **Zarządzaj dodatkami**
3. Porada: spowoduje to otwarcie aplikacji Outlook w sieci Web.
4. W obszarze Zarządzanie dodatkami w kolumnie włączone **Usuń zaznaczenie pola wyboru dodatku**  , który chcesz wyłączyć.
5. Włączanie lub wyłączanie dodatków

Porada: Aby ponownie włączyć dodatek, zaznacz to pole wyboru.
