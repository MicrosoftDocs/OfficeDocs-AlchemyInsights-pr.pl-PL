# <a name="create-or-edit-an-outlook-profile"></a>Tworzenie lub edytowanie profilu programu Outlook

Aby utworzyć nowy profil programu Outlook, edytować profil lub zmienić nazwę profilu, wykonaj następujące czynności.
  
1. W programie Outlook wybierz **File** pozycję \> **Ustawienia kont** plików \> **Zarządzaj profilami**.
    
2. W oknie dialogowym **Konfiguracja poczty** wybierz pozycję **Pokaż profile**.
    
3. Wykonaj jeden z następujących kroków:
    
  - Aby utworzyć nowy profil, wybierz pozycję **Dodaj**.
    
  - Aby edytować profil lub zmienić jego nazwę, wybierz pozycję **Właściwości**.
    
Jeśli masz problemy z tworzeniem lub edytowaniem profilu programu Outlook, możesz uruchomić [Narzędzie profil programu Outlook](https://aka.ms/SaRA-OutlookSetupProfile) na komputerze, którego dotyczy problem. Sprawdzi to, a gdy wszystko będzie gotowe, utworzy profil programu Outlook z adresem e-mail.
