# <a name="change-the-font-size-in-a-message"></a>Zmienianie rozmiaru czcionki w wiadomości

1. Jeśli wiadomość została już wpisana, zaznacz tekst.
2. Wybierz ikonę  **formatu** poniżej wiadomości (jest to pierwsza ikona).
3. Wybierz ikonę  **rozmiar czcionki**  nad wiadomością (jest to ikona siódma).
4. Wybierz rozmiar.
