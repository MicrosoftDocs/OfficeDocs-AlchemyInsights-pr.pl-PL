# <a name="submit-powerapps-or-microsoft-flow-support-requests"></a>Przesyłać zlecenia pomocy technicznej PowerApp lub Flow firmy Microsoft

Jeśli jesteś **PowerApp** lub klienta **Microsoft przepływu** Prześlij bilet za pośrednictwem nowej [platformy zasilania Centrum administracyjnego](https://admin.powerplatform.microsoft.com/support?newTicket&product=15819). Nowy portal zapewnia bardziej bezpośrednią ścieżkę do asystowana pomoc techniczna jest zoptymalizowany pod kątem PowerApp i innych produktów platformy zasilania.
