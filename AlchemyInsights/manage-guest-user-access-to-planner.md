# <a name="manage-guest-user-access-to-planner"></a>Zarządzanie dostępem użytkowników będących gośćmi do usługi Planner

Użytkownicy będący gośćmi mogą tworzyć i usuwać zasobniki oraz zadania, edytować pola zadań i edytować nazwę planu. Ponadto użytkownicy będący gośćmi otrzymają również niektóre powiadomienia skojarzone z Twoim planem. Poniżej przedstawiono kilka często zaproszonych tematów dotyczących użytkowników będących Gośćmi w usłudze Planner:

- Kompilacje usługi Planner w usłudze [365 Microsoft Access Groups — gość](https://support.office.com/article/Adding-guests-to-Office-365-Groups-bfc7a840-868f-4fd6-a390-f347bf51aff6). 

- Dowiedz się więcej o tym, jak dodawać użytkowników będących gośćmi do planu, jak użytkownicy gość mogą opuścić plan, a także jak użytkownicy gość mogą wykonywać zadania w usłudze Planner w usłudze [dostęp Gości w usłudze Microsoft Planner](https://support.office.com/article/Guest-access-in-Microsoft-Planner-cc5d7f96-dced-4da4-ab62-08c72d9759c6).
