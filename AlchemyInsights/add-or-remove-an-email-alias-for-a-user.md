# <a name="add-or-remove-an-email-address-for-a-user"></a>Dodawanie lub usuwanie adresu e-mail użytkownika

[Aby](https://portal.office.com/AdminPortal/Home#/AssistedGuide/addemailoptions) rozpocząć, wykonaj następujące kroki w centrum administracyjnym usługi Microsoft 365.

 **Uwaga:** Użytkownik może mieć więcej niż jeden adres e-mail, ale dodatkowe *aliasy e-mail* mogą być używane tylko do odbierania wiadomości e-mail. Jeśli chcesz wysłać wiadomość e-mail z aliasem, utwórz [udostępnioną skrzynkę pocztową](https://docs.microsoft.com/microsoft-365/admin/email/create-a-shared-mailbox). W ten sposób otrzymasz alias do wysyłania i odbierania poczty.
