# <a name="microsoft-365-guest-sharing-settings-reference"></a>Informacje dotyczące ustawień udostępniania dla gości na platformie Microsoft 365

Poniższy artykuł zawiera informacje na temat różnych ustawień, które mogą mieć wpływ na udostępnianie osobom spoza organizacji w ramach obciążeń platformy Microsoft 365: Teams, Grupy usługi Office 365, SharePoint i OneDrive. Te ustawienia znajdują się w centrach administracyjnych usługi Azure Active Directory, platformy Microsoft 365, usługi Teams i programu SharePoint.
