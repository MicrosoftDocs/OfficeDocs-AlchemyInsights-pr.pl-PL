# <a name="proxyaddress-incorrect"></a>Nieprawidłowa adres proxy

Gdy obiekt jest synchronizowany z usługą Azure AD, wartości określone w proxyAddresses atrybut w usłudze Active Directory są porównywane z regułami usługi Azure AD, a następnie proxyAddresses atrybut jest wypełniany w usłudze Azure AD. W związku z tym wartości proxyAddresses atrybut dla obiektu w usłudze Active Directory może nie być taka sama jak wartości proxyAddresses atrybut w usłudze Azure AD.
  
Aby dowiedzieć się więcej o wypełnianiu adresów proxy, zobacz [Jak atrybut proxyAddress jest wypełniany w usłudze Azure AD](https://support.microsoft.com/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad).
