# <a name="proxyaddress-incorrect"></a>ProxyAddress niepoprawne

Gdy obiekt jest zsynchronizowany z usługą Azure AD, wartości, które są określone w atrybucie proxyAddresses w usłudze Active Directory są porównywane z regułami usługi Azure AD, a następnie atrybut proxyAddresses jest wypełniona w usłudze Azure AD. W związku z tym wartości atrybutu proxyAddresses dla obiektu w usłudze Active Directory mogą nie być takie same, jak wartości atrybutu proxyAddresses w usłudze Azure AD.
  
Aby dowiedzieć się więcej na temat sposobu ProxyAddress jest wypełniona, zobacz [jak atrybut ProxyAddress jest wypełniona w usłudze Azure AD](https://support.microsoft.com/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad).
