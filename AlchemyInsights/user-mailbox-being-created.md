# <a name="your-users-mailbox-is-being-created"></a>Trwa Tworzenie skrzynki pocztowej użytkownika

Może upłynąć do godziny, zanim nowa skrzynka pocztowa będzie gotowa do użycia.
  
Jeśli skrzynka pocztowa użytkownika jest "przygotowywana" przez ponad godzinę, możesz spróbować usunąć i ponownie utworzyć użytkownika albo skorzystać z opcji "Pozwól nam zadzwonić do Ciebie", aby porozmawiać z inżynierem pomocy technicznej.
