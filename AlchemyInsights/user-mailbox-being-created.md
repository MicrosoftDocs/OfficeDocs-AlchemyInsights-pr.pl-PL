# <a name="your-users-mailbox-is-being-created"></a>Utworzono skrzynkę pocztową użytkownika

Może upłynąć do godziny, zanim nowa skrzynka pocztowa będzie gotowa do użycia.
  
Jeśli skrzynka pocztowa użytkownika "przygotowuje się" przez ponad godzinę, możesz spróbować usunąć i odtworzyć użytkownika lub użyć opcji "Zadzwoń do Ciebie", aby porozmawiać z inżynierem pomocy technicznej.
