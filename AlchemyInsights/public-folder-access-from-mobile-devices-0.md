# <a name="public-folder-access-from-mobile-devices"></a>Dostęp do folderów publicznych z urządzeń przenośnych

Foldery publiczne nie są dostępne na urządzeniach przenośnych.
  
Jeśli chcesz, aby funkcje były podobne do folderów publicznych i uzyskiwać do nich dostęp z urządzeń przenośnych, zobacz [więcej informacji o grupach Microsoft 365](https://support.office.com/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2) w celu uzyskania lepszej alternatywy.
