# <a name="publishing-infrastructure-with-a-communication-site"></a>Infrastruktura publikowania z witryną komunikacyjną


Włączenie funkcji Infrastruktury publikowania programu SharePoint Server nie jest **obsługiwane w** witrynach komunikacyjnych. 
  
Aby uzyskać więcej informacji, zobacz [Przenoszenie z witryn publikowania do witryn komunikacyjnych](https://docs.microsoft.com/sharepoint/publishing-sites-classic-to-modern-experience).
