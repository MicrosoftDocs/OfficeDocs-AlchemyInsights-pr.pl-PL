# <a name="publishing-infrastructure-with-a-communication-site"></a>Publikowanie infrastruktury z witryną komunikacyjną


Włączenie funkcji publikowania infrastruktury serwera programu SharePoint **nie jest obsługiwana** w witrynach komunikacyjnych. 
  
Aby uzyskać więcej informacji, zobacz [przenoszenie z witryn publikowania do witryn komunikacyjnych](https://docs.microsoft.com/sharepoint/publishing-sites-classic-to-modern-experience).
