# <a name="publishing-infrastructure-with-a-communication-site"></a>Publikowanie infrastruktury za pomocą witryny do komunikacji


Włączanie funkcji infrastruktury publikowania w programie SharePoint Server **nie jest obsługiwane** w witrynach komunikacyjnych. 
  
Aby uzyskać więcej informacji, zobacz [przechodzenie z witryn publikowania do witryn komunikacji](https://docs.microsoft.com/sharepoint/publishing-sites-classic-to-modern-experience).
