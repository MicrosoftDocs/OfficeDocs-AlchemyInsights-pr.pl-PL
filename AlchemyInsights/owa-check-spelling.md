# <a name="use-your-browser-to-check-spelling"></a>Sprawdzanie pisowni za pomocą przeglądarki

W aplikacji Outlook w sieci Web nie ma opcji sprawdzania pisowni. Aby sprawdzić pisownię, musisz użyć przeglądarki internetowej. Sprawdzanie pisowni jest dostępne w programach Microsoft Edge, programie Internet Explorer 10 i nowszych oraz w bieżących wersjach przeglądarki Firefox, Chrome i Safari. Sprawdź opcje przeglądarki internetowej, aby dowiedzieć się więcej o tym, jak sprawdzić pisownię.
