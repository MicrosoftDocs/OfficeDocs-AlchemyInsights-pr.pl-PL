# <a name="software-updates-are-not-being-deployed"></a>Aktualizacje oprogramowania nie są wdrażane

Jeśli aktualizacje oprogramowania zostały skonfigurowane za pośrednictwem pierścienia aktualizacji systemu Windows 10, ale aktualizacje nie są wdrażane, spróbuj wykonać następujące czynności:  

- Należy rozważyć zmianę obsługi systemu Windows z **półrocznego** typu wydania kanału na bardziej rygorystyczny, częstszy typ wydania.
- Sprawdź okres odroczenia **aktualizacji jakości** i **funkcji**. Okres odroczenia może prowadzić do opóźnienia aktualizacji do 180 dni.
