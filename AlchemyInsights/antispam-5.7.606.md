# <a name="banned-sending-ip"></a>Zbanowany adres IP wysyłania

Podczas wysyłania poczty do usługi Microsoft 365 nadawca może wystąpić ten błąd. Nadawca musi odwiedzić https://sender.office.com i postępować zgodnie z instrukcjami.  Jeśli nie otrzymają e-maila weryfikacyjnego z portalu, poproś ich, aby sprawdzili folder spamu i/lub spróbujeli użyć innego adresu e-mail.
