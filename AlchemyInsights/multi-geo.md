# <a name="microsoft-365-multi-geo"></a>Microsoft 365 Multi-Geo

Dzięki usłudze Microsoft 365 Multi-Geo organizacja może rozszerzyć swoją obecność Microsoft 365 na wiele regionów geograficznych i/lub krajów w ramach Twojej istniejącej dzierżawy. Aby uzyskać więcej informacji, zobacz linki poniżej.

- [Microsoft 365 Multi-Geo](https://docs.microsoft.com/office365/enterprise/office-365-multi-geo)
- [Plan dla usługi Microsoft 365 Multi-Geo](https://docs.microsoft.com/office365/enterprise/plan-for-multi-geo)
- [Konfigurowanie wielu lokalizacji geograficznych](https://docs.microsoft.com/office365/enterprise/multi-geo-tenant-configuration)
