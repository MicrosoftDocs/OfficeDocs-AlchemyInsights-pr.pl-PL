# <a name="create-a-new-folder"></a>Utwórz nowy folder

1. W obszarze **folderów** w okienku nawigacji wybierz **Nowy folder**. 
    
2. Wpisz nazwę folderu i naciśnij klawisz Enter.
    
Aby dowiedzieć się więcej, zobacz [Tworzenie folderu w Hotmail](https://support.office.com/article/5fa8de74-3562-4729-ac1d-5599f470b25a).
