# <a name="microsoft-search-setup-guide"></a>Przewodnik konfiguracji usługi Microsoft Search

Aby skonfigurować i skonfigurować usługę [Microsoft Search,](https://go.microsoft.com/fwlink/?linkid=2153798) skorzystaj z przewodnika konfiguracji usługi Microsoft Search. Czy chcesz go pilotażować w grupie użytkowników, czy rozdysłać go wszystkim w organizacji. Przypiszesz administratorów wyszukiwania i edytorów wyszukiwania, a następnie dostosuj środowisko wyszukiwania dla użytkowników, dodając odpowiedzi i dodatkowe opcje, takie jak dodanie rozszerzenia Bing do przeglądarki Chrome lub ustawienie usługi Bing jako domyślnej wyszukiwarki.
