# <a name="does-visio-work-on-a-mac"></a>Czy program Visio działa na komputerze Mac?

Brak aplikacji klasycznej Visio dla komputerów Mac. Jednak tak długo, jak użytkownik, który potrzebuje aplikacji ma [przypisaną licencję visio online,](https://docs.microsoft.com/office365/admin/subscriptions-and-billing/assign-licenses-to-users?wt.mc_id=OfficeAdm_ClientDIA_Alchemy1799)mogą [wyświetlać, tworzyć i edytować diagramy w programie Visio Online](https://support.office.com/article/06f04845-91b8-4e8f-881f-a43c970735fc?wt.mc_id=OfficeAdm_ClientDIA_Alchemy1799) w przeglądarce sieci Web.
