# <a name="sharepoint-migration-throttling"></a>Ograniczanie migracji programu SharePoint

Usługa SharePoint Online używa ograniczania, aby zapewnić optymalną wydajność i niezawodność usługi SharePoint Online. Ograniczanie ogranicza liczbę akcji użytkownika lub współbieżnych połączeń (za pomocą skryptu lub kodu), aby zapobiec nadmiernemu zużyciu zasobów. Aby uzyskać więcej informacji, zobacz:

- [Podczas migracji występuje niska wydajność lub ograniczanie](https://docs.microsoft.com/sharepointmigration/sharepoint-online-and-onedrive-migration-speed#faq-and-troubleshooting)
