# <a name="create-users-in-hybrid-deployments"></a>Tworzenie użytkowników we wdrożeniach hybrydowych

Jeśli organizacja jest wdrożeniem hybrydowym między środowiskiem lokalnym i programem Microsoft 365, należy utworzyć wszystkie obiekty użytkowników w lokalnej usłudze Active Directory. Możesz utworzyć obiekty użytkowników w odpowiedniej jednostce organizacyjnej, aby były one synchronizowane z organizacją.
