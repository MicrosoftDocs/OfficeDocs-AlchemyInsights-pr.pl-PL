# <a name="create-users-in-hybrid-deployments"></a>Tworzenie użytkowników we wdrożeniach hybrydowych

Gdy organizacja jest wdrożeniem hybrydowym między środowiskiem lokalnym a usługą Microsoft 365, należy utworzyć wszystkie obiekty użytkownika w lokalnej usłudze Active Directory. Obiekty użytkownika są tworzą w odpowiedniej jednostce organizacyjnej, aby były synchronizowane z organizacją.
