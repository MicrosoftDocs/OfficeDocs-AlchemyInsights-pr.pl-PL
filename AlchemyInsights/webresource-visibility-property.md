# <a name="webresource-visibility-property"></a>Właściwość Webresource Visibility

Należy wybrać opcję **Widoczna domyślnie,** aby zasób sieci web był wyświetlany w formularzu i był dostępny dla użytkowników. Aby uzyskać więcej informacji, zobacz [Właściwości zasobów sieci Web dla formularzy aplikacji opartych na modelu](https://docs.microsoft.com/powerapps/maker/model-driven-apps/web-resource-properties-legacy).
