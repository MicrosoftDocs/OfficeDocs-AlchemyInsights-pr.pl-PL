# <a name="move-email-to-microsoft-365"></a>Przenoszenie poczty e-mail do usługi Microsoft 365

Dostępnych jest kilka opcji migracji poczty e-mail do usługi Microsoft 365. [Doradca migracji poczty usługi Microsoft 365](https://aka.ms/alchemyinsight-mailmigrationadvisor) pomaga wybrać najlepszy dla twojego środowiska, a także prowadzi użytkownika przez migrację poczty.
  
Aby zapoznać się z opcjami bez rozpoczynania migracji, zobacz [Decyduj o ścieżce migracji](https://docs.microsoft.com/Exchange/mailbox-migration/decide-on-a-migration-path).

Jeśli kwalifikujesz się, sprawdź również stronę [Adoptuj usługę Microsoft 365 z](https://www.microsoft.com/fasttrack/microsoft-365/office-365) programem FastTrack, aby uzyskać pomoc dotyczącą migracji.
