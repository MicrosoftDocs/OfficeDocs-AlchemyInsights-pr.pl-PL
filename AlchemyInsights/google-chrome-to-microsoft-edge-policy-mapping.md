# <a name="learn-about-mapping-between-google-chrome-policies-and-microsoft-edge-policies"></a>Informacje na temat mapowania między zasadami usługi Google Chrome a zasadami programu Microsoft Edge

Zasady usługi Google Chrome są mapowane na odpowiednie zasady programu Microsoft Edge obsługiwane w wersji 80 przeglądarki. Aby dowiedzieć się więcej, zobacz [mapowanie zasad programu Google Chrome na aplikację Microsoft Edge](https://go.microsoft.com/fwlink/?linkid=2141933).
