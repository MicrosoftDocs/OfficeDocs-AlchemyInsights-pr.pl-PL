# <a name="user-permissions-in-sharepoint"></a>Uprawnienia użytkownika w programie SharePoint

Po utworzeniu witryny programu SharePoint, można zapewnić lub ograniczyć dostęp użytkownika do witryny lub jej zawartości. Najprostszym sposobem pracy z uprawnieniami jest przy użyciu poziomów [domyślnych grup](https://support.office.com/article/default-sharepoint-groups-13bb2b6b-dd8c-447e-b71b-0e4bb9efe1d3) i uprawnień, które obejmują najbardziej typowe scenariusze. Ale jeśli trzeba, można ustawić więcej uprawnienia szczegółowe poza domyślnych poziomów. Aby uzyskać więcej informacji zobacz [Opis poziomów uprawnień w programie SharePoint](https://docs.microsoft.com/sharepoint/understanding-permission-levels).

Aby uzyskać więcej informacji dotyczących zarządzania określonych poziomów uprawnień, odwiedź poniższych łączy:

- [Edytowanie i zarządzanie uprawnieniami dla listy programu SharePoint lub biblioteki](https://support.office.com/article/customize-permissions-for-a-sharepoint-list-or-library-02d770f3-59eb-4910-a608-5f84cc297782)

- [Tworzenie i zarządzanie grupami programu SharePoint](https://support.office.com/article/create-and-manage-sharepoint-groups-b1e3cd23-1a78-4264-9284-87fed7282048)

- [Jak utworzyć i edytować poziomy uprawnień](https://docs.microsoft.com/sharepoint/how-to-create-and-edit-permission-levels)

- [Konfigurowanie i zarządzanie żądaniami dostępu](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3)

- [Należy zmienić uprawnienia do podfolderu](https://support.office.com/article/change-the-permissions-on-a-subfolder-5427bd7c-f20a-4f75-8cf2-5359dd45a1a6)

- [Udostępniać pliki programu SharePoint lub foldery w usłudze Office 365](https://support.office.com/article/share-sharepoint-files-or-folders-1fe37332-0f9a-4719-970e-d2578da4941c)

- [Udostępnianie witryny](https://support.office.com/article/share-a-site-958771a8-d041-4eb8-b51c-afea2eae3658)
