# <a name="restore-a-previous-file-version"></a>Przywracanie poprzedniej wersji pliku

Historia wersji online umożliwia wyświetlanie i przywracanie starszych wersji plików w usłudze OneDrive lub w programie SharePoint. Historia wersji działa ze wszystkimi typami plików, w tym plikami PDF, CAD, zdjęciami i filmami. Aby przywrócić poprzednią wersję pliku:
  
1. Otwórz bibliotekę dokumentów programu SharePoint lub lokalizację OneDrive.
    
2. Kliknij po lewej stronie pliku, aby go zaznaczyć, a następnie kliknij pozycję **historia wersji**. Jeśli nie widzisz historii wersji na pasku poleceń, kliknij więcej ( **...** ). 
    
3. Kliknij wersję, którą chcesz wyświetlić, lub kliknij przycisk więcej ( **...** ) po prawej stronie wersji, a następnie kliknij przycisk **Przywróć**.
