# <a name="submit-microsoft-cloud-for-healthcare-support-requests"></a>Przesyłanie wniosków o pomoc techniczną firmy Microsoft w chmurze

Microsoft Cloud dla klienta z placówką o usłudze Power admin Prześlij swój bilet za pośrednictwem nowego [Centrum administracyjnego platformy Power platform.](https://admin.powerplatform.microsoft.com/support?newTicket&product=Flow)

Nowy portal oferuje bardziej bezpośrednią ścieżkę do pomocy technicznej zoptymalizowanej pod kątem usługi Microsoft Cloud dla służb opieki zdrowotnej.
