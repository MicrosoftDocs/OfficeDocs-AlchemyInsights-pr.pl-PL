# <a name="install-project-professional"></a>Instalowanie programu Project Professional

Zobacz [Instalowanie programu Project,](https://support.office.com/article/install-project-7059249b-d9fe-4d61-ab96-5c5bf435f281) aby dowiedzieć się, jak zainstalować wersję subskrypcji usługi Project Online oraz wersję niespłacadniczą programu Project 2019, 2016 lub 2013. 

W przypadku programu Project 2010 lub 2007 wykonaj kroki opisane w [pakiecie Office 2010](https://support.office.com/article/install-office-2010-1b8f3c9b-bdd2-4a4f-8c88-aa756546529d) lub [Office 2007](https://support.office.com/article/install-office-2007-88a8e329-3335-4f82-abb2-ecea3e319657). 

Projekt jest sprzedawany jako samodzielna aplikacja, więc jeśli jeszcze go nie masz, [Kup lub wypróbuj program Project](https://products.office.com/project).
