# <a name="links-in-outlookcom-dont-work-in-firefox"></a>Linki w Outlook.com nie działają w programie Firefox

Linki w Outlook.com wiadomościach e-mail nie działają, gdy są wyświetlane w przeglądarkach Firefox. Program Mozilla [pracuje nad rozwiązaniem](https://go.microsoft.com/fwlink/p/?linkid=2001502&amp;clcid=0x409) dotyczącym przeglądarki Firefox, ale do tego:
  
- Użyj przeglądarki [Microsoft Edge](https://go.microsoft.com/fwlink/p/?linkid=2001503&amp;clcid=0x409) lub innej przeglądarki sieci Web.
- Wyświetlanie wiadomości e-mail w okienku odczytu. Aby uzyskać więcej informacji, zobacz [Zmienianie wyglądu skrzynki pocztowej programu Outlook.com](https://support.office.com/article/b41c2ecb-f23c-42b3-b7f8-659646d5e58c?wt.mc_id=Office_Outlook_com_Alchemy).
