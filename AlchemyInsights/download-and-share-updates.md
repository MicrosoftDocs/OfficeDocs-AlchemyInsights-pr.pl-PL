# <a name="download-and-share-updates"></a>Pobieranie i udostępnianie aktualizacji

Zamiast pobierać aktualizacje indywidualnie, tryb pobierania optymalizacji dostarczania umożliwia udostępnianie między wieloma komputerami. Aby uzyskać więcej informacji, zobacz [Optymalizacja dostarczania dla aktualizacji systemu Windows 10](https://docs.microsoft.com/windows/deployment/update/waas-delivery-optimization).  

Dostępne tryby pobierania obejmują:  
- Tylko HTTP  
- Lan  
- Grupa  
- Internet  
- Simple (Proste)  
- Obwodnica
