# <a name="windows-fresh-start"></a>Nowy początek systemu Windows

Akcja urządzenia Windows Fresh Start oferuje alternatywę dla akcji Czyszczenie urządzenia i Wycofanie urządzenia wraz z opcją zachowania stanu rejestracji urządzenia wraz z aplikacjami i ustawieniami folderu macierzystego użytkownika.

Aby uzyskać więcej informacji, zobacz [Resetowanie urządzeń z systemem Windows 10 za pomocą usługi Intune za pomocą funkcji Fresh Start](https://docs.microsoft.com/intune/device-fresh-start).
