# <a name="how-to-customize-permissions-for-a-sharepoint-list-or-library"></a>Jak dostosować uprawnienia do listy lub biblioteki programu SharePoint

Jako administrator lub właściciel biblioteki, listy lub ankiety możesz zmienić uprawnienia, aby umożliwić jednej grupie osób uzyskiwanie dostępu do potrzebnych danych w celu ograniczenia możliwości innych.

[Dostosowywanie uprawnień do listy lub biblioteki programu SharePoint](https://support.office.com/article/customize-permissions-for-a-sharepoint-list-or-library-02d770f3-59eb-4910-a608-5f84cc297782)
