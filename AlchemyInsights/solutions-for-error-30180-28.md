# <a name="solutions-for-error-30180-28"></a>Rozwiązania dla błędu 30180-28

[Błąd 30180-28](https://support.office.com/article/47ae453b-677c-412f-9a21-6766555ff4de?wt.mc_id=Alchemy_ClientDIA) może wystąpić podczas aktualizacji pakietu Office, jeśli masz słabe połączenie internetowe. Oto kilka rzeczy, możesz spróbować rozwiązać problem. 
  
- Uruchom ponownie komputer i następnie [aktualizacji pakietu Office](https://support.office.com/article/2ab296f3-7f03-43a2-8e50-46de917611c5?wt.mc_id=Alchemy_ClientDIA).
    
- Wykonaj powyższe czynności, ale używając połączenia przewodowego podczas aktualizacji pakietu Office.
