# <a name="inbox-not-loading-with-firefox"></a>Skrzynka odbiorcza nie ładuje z Firefox

Niektóre dodatki w przeglądarce Firefox może uniemożliwić skrzynce odbiorczej Hotmail załadunku.
  
Odwiedź stronę wsparcia Firefox, aby dowiedzieć się, jak [wyłączyć lub usunąć dodatki](https://support.mozilla.org/kb/disable-or-remove-add-ons).
