# <a name="give-users-access-to-sharepoint-site"></a>Udzielanie użytkownikom dostępu do witryny programu SharePoint

Funkcja żądania dostępu umożliwia użytkownikom żądanie dostępu do zawartości, do których obecnie nie mają uprawnień. Jako właściciel witryny możesz skonfigurować tę funkcję do wysyłania poczty, gdy ktoś zażąda dostępu do witryny. Aby uzyskać więcej informacji, odwiedź stronę [Konfigurowanie żądań dostępu i zarządzanie nimi.](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3)

Oto kilka typowych linków dotyczących żądań dostępu:

- [Odmowa dostępu" do listy żądań programu Access lub "Zatwierdzenie żądania nie powiodło się" podczas przetwarzania oczekującego żądania w usłudze SharePoint Online](https://docs.microsoft.com/sharepoint/support/sharing-and-permissions/request-approval-failed)

- [Zmienianie ustawień żądań dostępu](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3#bk_enableallow)

- [Zatwierdzanie lub odrzucanie żądań dostępu dla witryny](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3#__toc374462558)

- [Zarządzanie zaproszeniami dla użytkowników zewnętrznych (tylko sharepoint online)](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3#__toc334189260)

- [Wyślij ponownie zaproszenie](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3#__toc374462560)
