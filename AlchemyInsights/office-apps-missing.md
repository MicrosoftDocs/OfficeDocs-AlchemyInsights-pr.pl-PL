# <a name="microsoft-365-apps-missing"></a>Brak aplikacji microsoft 365

Jeśli nie możesz zlokalizować aplikacji pakietu Office na komputerze po zainstalowaniu lub aktualizacji, zobacz [Nie można znaleźć aplikacji pakietu Office w systemie Windows 10, Windows 8 lub Windows 7?](https://support.office.com/article/Can-t-find-Office-applications-in-Windows-10-Windows-8-or-Windows-7-907ce545-6ae8-459b-8d9d-de6764a635d6). Jeśli nie możesz znaleźć skrótów pakietu Office w menu Start lub w obszarze **Wszystkie aplikacje**, zobacz [Brak aplikacji usługi Microsoft 365 we wszystkich aplikacjach w systemie Windows 10](https://support.office.com/article/office-apps-are-missing-from-all-apps-on-windows-10-5bc123f6-655d-4736-ad61-b0b9d1cde5bc). 

Spróbuj [naprawić aplikację pakietu Office,](https://support.office.com/article/repair-an-office-application-7821d4b6-7c1d-4205-aa0e-a6b40c5bb88b) jeśli skróty pakietu Office zniknęły po zalogowaniu się lub ponownym uruchomieniu komputera. 

**Uwaga:** Jeśli brakuje aplikacji usługi Microsoft 365 do instalacji w portalu pakietu Office, przejdź do [witryny Rozwiązywanie problemów z instalowaniem pakietu Office](https://support.office.com/article/troubleshoot-installing-office-35ff2def-e0b2-4dac-9784-4cf212c1f6c2) i przejdź do **pakietu Office dla firm**Typowe  >  **problemy**Nie widzisz  >  **opcji instalowania pakietu Office**.
