# <a name="microsoft-stream-storage"></a>Magazyn strumieniowy Microsoft

Dzierżawy strumienia Microsoft Stream otrzymują stałą alokację 500 GB miejsca do magazynowania oraz dodatkową 0,5 GB miejsca do magazynowania na użytkownika licencjonowanego.
Aby uzyskać więcej informacji na temat przechowywania, zobacz:

- [Omówienie usługi Microsoft Stream Storage](https://docs.microsoft.com/stream/license-overview#storage)
- [Dodawanie miejsca do usługi Microsoft Stream](https://docs.microsoft.com/stream/storage-add-on)
- [Przydziały i ograniczenia w usłudze Microsoft Stream](https://docs.microsoft.com/stream/quotas-and-limitations)
