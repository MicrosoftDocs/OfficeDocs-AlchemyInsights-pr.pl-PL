# <a name="move-a-folder"></a>Przenoszenie folderu

Można zmienić kolejność folderów w folderze **Ulubione**, ale nie można zmienić kolejność folderów w **folderach** lub uczynić podfolderem folderu. Jeśli przypadkowo przenieść folder do innego folderu, Utwórz nowy folder i skopiuj adres e-mail z podfolderu w nowym folderze.
  
Aby dowiedzieć się więcej, przeczytaj [pracy z folderami wiadomości w Hotmail](https://support.office.com/article/6bb0723a-f39f-4a8d-bb3f-fab5dcc2510a).
