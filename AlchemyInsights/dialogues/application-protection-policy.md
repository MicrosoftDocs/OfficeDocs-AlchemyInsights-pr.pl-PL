# <a name="application-protection-policy-requirements"></a>Wymagania dotyczące zasad ochrony aplikacji

Wymagania dotyczące zasad ochrony aplikacji:

- Użytkownik ma licencję usługi Intune lub EMS.
- Użytkownik należy do grupy docelowej przez zasady ochrony aplikacji.
- Tylko jeden użytkownik firmowy jest zalogowany do aplikacji chronionych na urządzeniu.
- [Aplikacja wdrożyła zestaw SDK usługi Intune. Aby uzyskać listę aplikacji, które obsługują zestaw SDK, zobacz aplikacje chronione przez usługę Microsoft Intune.](https://docs.microsoft.com/mem/intune/apps/apps-supported-intune-apps)
