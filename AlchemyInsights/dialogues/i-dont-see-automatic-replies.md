# <a name="i-dont-see-automatic-replies"></a>Nie widzę odpowiedzi automatycznych

Jeśli po wybraniu pliku nie widzisz opcji Odpowiedzi **automatyczne,** prawdopodobnie korzystasz z konta Gmail, Yahoo lub innego konta POP lub IMAP, które nie obsługuje funkcji Odpowiedzi automatyczne programu Outlook. Możesz skonfigurować regułę, która będzie odpowiadać na przychodzące wiadomości, ale tylko jeśli pozostawisz program Outlook uruchomiony.
