# <a name="set-up-safe-attachment-policies-in-microsoft-defender-for-office-365"></a>Konfigurowanie zasad bezpiecznego załącznika w usłudze Microsoft Defender dla usługi Office 365

1. Przejdź do Centrum zgodności zabezpieczeń usługi [Office 365 &](https://go.microsoft.com/fwlink/p/?linkid=2077143) i wybierz **bezpieczne** załączniki zasad zarządzania  >    >  **zagrożeniami.**
2. Aby włączyć tę funkcję dla innych obciążeń pracą, wybierz pozycję Włącz usługę Microsoft Defender dla **usługi Office 365 dla usług SharePoint Online, OneDrive i Microsoft Teams.**
3. Aby utworzyć nowe zasady, wybierz **ikonę** Nowy ( **+** ).
4. Określ nazwę, opis i ustawienia zasad.
5. Wybierz **Zapisz**.
