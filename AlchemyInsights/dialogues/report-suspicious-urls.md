# <a name="report-suspicious-urls"></a>Zgłaszanie podejrzanych adresów URL

Aby przesłać podejrzane adresy URL, wykonaj następujące czynności:

1. W **wysuwaniu** Nowe przesyłanie wprowadź pełny adres URL wraz z protokołem (https://).
2. Jeśli wybierzesz opcję **Powinna być zablokowana,** określ, czy adres URL to **wyłudzanie** informacji, czy złośliwe **oprogramowanie.**
3. Wybierz **pozycję Prześlij.**
