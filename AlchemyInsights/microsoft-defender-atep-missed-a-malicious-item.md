# <a name="microsoft-defender-atp-missed-a-malicious-item"></a>Usługa Microsoft Defender ATP przegapiła złośliwy element

- Prześlij złośliwy plik do firmy Microsoft w celu analizy w [portalu Microsoft Defender Response Portal](https://www.microsoft.com/wdsi/filesubmission/). 
- Jeśli jesteś zalogowany, znajdź historię przesyłania pliku na portalu [Microsoft Defender Response Portal](https://www.microsoft.com/wdsi/submissionhistory).
- Aby uzyskać więcej informacji, zobacz [Przesyłanie plików do analizy](https://docs.microsoft.com/windows/security/threat-protection/intelligence/submission-guide).
