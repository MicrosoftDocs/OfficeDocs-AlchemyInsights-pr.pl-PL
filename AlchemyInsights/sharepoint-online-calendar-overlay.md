# <a name="sharepoint-online-calendar-overlay"></a>Nakładka kalendarza w usłudze SharePoint Online

Możesz nałożyć inny Kalendarz programu SharePoint, o ile oba kalendarze programu SharePoint znajdują się w tym samym zbiorze witryn. Możliwość korzystania z nakładki kalendarza w kalendarzu programu SharePoint Online lub programu SharePoint Server, który jest pobierany z usługi Exchange Online, nie jest obsługiwana.

SharePoint Online oferuje nowoczesne środowisko pracy z kalendarzami w grupie Microsoft 365. Aby dowiedzieć się więcej, zobacz [Korzystanie z składnika Web Part kalendarza grupy](https://support.microsoft.com/en-us/office/use-the-group-calendar-web-part-eaf3c04d-5699-48cb-8b5e-3caa887d51ce).
