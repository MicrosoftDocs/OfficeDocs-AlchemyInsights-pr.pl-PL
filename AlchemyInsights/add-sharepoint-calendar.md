# <a name="add-a-sharepoint-calendar"></a>Dodawanie kalendarza programu SharePoint

Kalendarze programu SharePoint to klasyczne typy list, które czasami są mylone z kalendarzami grupowymi usługi Microsoft 365.
 
Aby utworzyć kalendarz programu SharePoint, wykonaj następujące czynności:
 
1.  Wybierz ikonę **Biegi ustawień,** a następnie wybierz pozycję **Dodaj aplikację**.
2.  W polu wyszukiwania **Znajdź aplikację** wpisz **pozycję Kalendarz** i naciśnij klawisz Enter.
3.  Wybierz aplikację o nazwie **Kalendarz**, a następnie wprowadź nazwę kalendarza.

Aby uzyskać więcej informacji na temat grup programu Microsoft 365, zobacz [Dowiedz się więcej o grupach usługi Microsoft 365](https://support.office.com/article/Learn-about-Office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2).
