# <a name="setting-up-a-hybrid-environment"></a>Konfigurowanie środowiska hybrydowego

Aby umożliwić użytkownikom bezproblemową współpracę bez względu na to, czy ich dane są przechowywane na serwerach lokalnych lub w usłudze Microsoft 365, może być konieczne skonfigurowanie środowiska hybrydowego. Aby uzyskać przegląd opcji, zapoznaj się [z integracją usługi Microsoft 365 ze środowiskami lokalnymi.](https://docs.microsoft.com/office365/enterprise/office-365-integration)
