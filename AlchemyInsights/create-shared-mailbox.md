# <a name="create-a-shared-mailbox"></a>Tworzenie udostępnionej skrzynki pocztowej

Współdzielonej skrzynki pocztowej umożliwia więcej niż jedna osoba, wysyłanie i odbieranie wiadomości e-mail dla danego aliasu, takich jak sales@contoso.com, bez przypisywania innej licencji. Aby utworzyć współdzielonej skrzynki pocztowej, przejdź do [Udostępnionych przewodnik E-mail](https://portal.office.com/adminportal/home).
  
Zobacz [Tworzenie udostępnionej skrzynki pocztowej](https://support.office.com/article/Create-a-shared-mailbox-871a246d-3acd-4bba-948e-5de8be0544c9) dla porady i więcej informacji o korzystaniu z udostępnionej skrzynki pocztowej.
