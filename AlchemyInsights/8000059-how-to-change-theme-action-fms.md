# <a name="how-to-change-your-theme"></a>Jak zmienić motyw

Aby zmienić kompozycję w programie Outlook w sieci web lub Hotmail, przejdź do **ustawień**![ustawienia](media/f4b2e798-fff1-4a14-931f-5677a4543b58.png) i wybierz motyw **tematu**. Aby wyświetlić wszystkie dostępne tematy, wybierz **Widok wszystkich** **tematu**.
