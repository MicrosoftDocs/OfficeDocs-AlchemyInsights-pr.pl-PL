# <a name="enable-or-disable-forms-pro"></a>Włączanie lub wyłączanie formularzy Pro

Jeśli próbujesz włączyć program Microsoft Forms Pro, jest ona dołączona do licencji [dla przedsiębiorstw Dynamics 365.](https://docs.microsoft.com/forms-pro/purchase#purchase-microsoft-forms-pro-for-users-in-a-dynamics-365-tenant) Alternatywnie administrator może kupić [licencję Microsoft Forms Pro](https://docs.microsoft.com/forms-pro/purchase#purchase-microsoft-forms-pro-for-users-in-a-dynamics-365-tenant). Można również [usunąć licencję Microsoft Forms Pro](https://docs.microsoft.com/forms-pro/purchase#disable-microsoft-forms-pro-for-a-user-1) od użytkownika.
