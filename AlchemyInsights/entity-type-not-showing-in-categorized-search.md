# <a name="entity-type-not-showing-in-search-results"></a>Typ jednostki nie jest wyświetlany w wynikach wyszukiwania

Dla typu jednostki do pokazania w wyszukiwania skategoryzowane wyszukiwania lub przydatność w interfejsie Unified Dynamics 365 należy najpierw dodać do modułu aplikacji.

Uwaga: Podczas wyszukiwania załączników w wyszukiwaniu przydatności, encji załącznika należy dodać do modułu aplikacji nawet jeśli rekord nadrzędny jest, co jest widoczne w wynikach.
