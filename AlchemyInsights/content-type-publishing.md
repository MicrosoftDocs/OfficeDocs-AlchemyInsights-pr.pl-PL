# <a name="content-type-publishing"></a>Publikowanie typu zawartości

Jeśli chcesz, aby typy zawartości były dostępne w całej organizacji dla wszystkich użytkowników, najlepiej jest publikować we wszystkich zbiorach witryn. Aby opublikować typ zawartości, zobacz [Publikowanie typu zawartości z centrum publikowania zawartości](https://support.office.com/article/publish-a-content-type-from-a-content-publishing-hub-58081155-118d-4e7a-9cc5-d43b5dbb7d02).
