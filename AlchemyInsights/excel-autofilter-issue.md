# <a name="excel-autofilter-issue"></a>Problem Autofiltr programu Excel

Przy użyciu programu Excel 2016, jeśli Twój rozwijanej Autofiltru nie pokazuje wszystkie dostępne pola wyboru lub pola wyboru nie są clickable, może to ze względu na znany problem w starszych wersjach programu Excel. 
  
Ten problem jest już ustalona w **wersji 1707 (8326.2073)** lub nowszym. Aby natychmiast uzyskać tę aktualizację, Otwórz program Excel i wybierz **plik** \> **konta** \> **Opcje aktualizacji** \> **Aktualizuj teraz**.
