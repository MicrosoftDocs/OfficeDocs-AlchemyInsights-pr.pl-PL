# <a name="cant-download-files-in-chrome"></a>Nie można pobrać plików w przeglądarce Chrome

**Problem** Wiemy o problemie uniemożliwiającym niektórym klientom pobieranie plików podczas korzystania z przeglądarki Google Chrome. 

**Obejście problemu** Użyj przeglądarki [Microsoft Edge](https://www.microsoft.com/windows/microsoft-edge) lub innej przeglądarki sieci Web.
Dowiedz się więcej na temat [pobierania plików i Outlook.com załączników](https://support.office.com/article/8d7c1ea7-4e5f-44ce-bb6e-c5fcc92ba9ab?wt.mc_id=Office_Outlook_com_Alchemy).
