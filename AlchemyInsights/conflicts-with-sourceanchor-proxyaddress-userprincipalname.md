# <a name="conflicts-with-sourceanchor-proxyaddress-userprincipalname"></a>Konflikty z SourceAnchor, ProxyAddress, UserPrincipalName

Jeśli podczas synchronizacji są pojawiały się błędy, takie jak "Zsynchronizowany obiekt z tym samym serwerem proxy lub Nazwa użytkownika istnieje w katalogu", zobacz [Diagnozowanie i korygowanie zduplikowanych błędów synchronizacji atrybutów](https://docs.microsoft.com/azure/active-directory/hybrid/how-to-connect-health-diagnose-sync-errors).

Należy również rozważyć włączenie odporności zduplikowanych atrybutów. Aby uzyskać więcej informacji, zobacz [Synchronizacja tożsamości i odporność atrybutów duplikatów](https://aka.ms/duplicateattributeresiliency).
