# <a name="convert-a-distribution-group-to-an-microsoft-365-group"></a>Konwertowanie grupy dystrybucyjnej na grupę usługi Microsoft 365

Grupy dystrybucyjne można konwertować na grupy usługi Microsoft 365, jeśli spełniają określone kryteria. Aby przekonwertować grupę lub rozwiązać problemy z konwersją, zobacz [Uaktualnianie list dystrybucyjnych do grup programu Microsoft 365 w programie Outlook](https://docs.microsoft.com/office365/admin/manage/upgrade-distribution-lists).
