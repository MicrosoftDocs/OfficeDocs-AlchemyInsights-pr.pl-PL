# <a name="upload-process-overview-in-microsoft-stream"></a>Omówienie procesu przekazywania w usłudze Microsoft Stream

Za każdym razem, gdy przesyłasz film, usługa Microsoft Stream automatycznie koduje go do wielu szybkości transmisji bitów, dzięki czemu wideo można przesyłać strumieniowo przy użyciu adaptacyjnego przesyłania strumieniowego szybkości transmisji bitów. Usługa Microsoft Stream optymalizuje szybkość transmisji i jakość odtwarzanego wideo na podstawie dostępnej przepustowości widza.

- [Omówienie procesu przekazywania w usłudze Microsoft Stream](https://docs.microsoft.com/stream/upload-process-overview)
