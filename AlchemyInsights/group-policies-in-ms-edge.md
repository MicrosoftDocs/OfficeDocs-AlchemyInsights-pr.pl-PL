# <a name="group-policies-in-microsoft-edge"></a>Zasady grupy w programie Microsoft Edge

Aby określić sposób uruchamiania przeglądarki w organizacji, użyj [zasad grupy](https://go.microsoft.com/fwlink/?linkid=2134623) dostępnych w programie Microsoft Edge (w wersji 77 lub nowszej).
