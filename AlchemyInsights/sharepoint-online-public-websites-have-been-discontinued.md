# <a name="sharepoint-online-public-websites-have-been-discontinued"></a>Publiczne witryny sieci Web usługi SharePoint Online zostały wycofane

Publiczne witryny sieci Web usługi SharePoint Online zostały wycofane.

Jeśli utworzono publiczną witrynę sieci Web usługi SharePoint Online, można było kontynuować korzystanie z niej do 31 marca 2018 r. Jeśli chcesz nadal mieć publiczną witrynę sieci Web, musisz utworzyć nową z nowym dostawcą. Zawartość publicznej witryny sieci Web usługi SharePoint Online została usunięta.

Aby uzyskać informacje o wycofaniu publicznej witryny sieci Web, zobacz [Szczegółowe informacje o procesie wycofywania funkcji](https://go.microsoft.com/fwlink/?linkid=866980).
