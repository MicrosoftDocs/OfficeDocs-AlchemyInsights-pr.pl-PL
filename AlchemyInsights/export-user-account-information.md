# <a name="export-user-account-information"></a>Eksportowanie informacji o koncie użytkownika

Aby wyeksportować konto użytkownika informacji, w tym informacje dotyczące licencji, zasady haseł, zablokowanych kont, stan synchronizacji AAD i więcej, przejdź do **użytkowników** > [aktywnych użytkowników](https://go.microsoft.com/fwlink/p/?linkid=834822) strony, a następnie wybierz polecenie **Eksportuj użytkowników**.
