# <a name="block-pop-up-windows-in-microsoft-edge"></a>Blokowanie wyskakujących okienek w przeglądarce Microsoft Edge

1. Przejdź do **... (Ustawienia i nie tylko)**  >  **Ustawienia**  >  **Uprawnienia witryny**.
2. Wybierz pozycję **wyskakujące okienka i przekierowania**.
3. Włączanie **bloku**.
