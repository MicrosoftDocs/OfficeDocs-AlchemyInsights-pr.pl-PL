# <a name="resolving-excel-clipboard-error"></a>Rozwiązywanie błędu w schowku programu Excel

Jeśli w programie Excel zostanie wyświetlony komunikat o błędzie "Wystąpił problem ze schowkiem, ale nadal można wkleić zawartość w tym skoroszycie, spróbuj wyłączyć funkcję Podgląd na żywo, wykonując następujące czynności:

1. W programie Excel kliknij **File**pozycję  >  **Opcje**pliku.
3. W oknie dialogowym **Opcje programu Excel** wybierz pozycję **Ogólne**.
4. W obszarze **Opcje interfejsu użytkownika**wyczyść pole wyboru **Włącz podgląd na żywo**.
5. Kliknij przycisk **OK**.
6. Uruchom ponownie program Excel.
