# <a name="how-to-access-outlookcom-while-traveling"></a>Jak uzyskać dostęp Hotmail podczas podróży

Dostęp Hotmail podczas podróży, upewnij się, że dane zabezpieczeń konta Microsoft - alternatywny adres e-mail adres lub numer telefonu skąd kody zabezpieczeń — jest aktualne. Aby dowiedzieć się więcej, zobacz [jak dostęp Hotmail podczas podróży](https://support.office.com/article/c44f16da-7156-4890-853c-286aafeda87e?wt.mc_id=Office_Outlook_com_Alchemy).
