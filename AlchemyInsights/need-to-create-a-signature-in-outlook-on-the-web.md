# <a name="need-to-create-an-email-signature"></a>Chcesz utworzyć podpis e-mail?

Jeśli masz konto programu Outlook lub konto programu Outlook w sieci Web (służbowej) można utworzyć podpis i automatycznie dodać go do wszystkich wiadomości wychodzących lub selektywnie dodać go do określonych wiadomości wychodzących. Pamiętaj, że podpis poczty e-mail jest inny w programach Outlook i Outlook w sieci Web. Aby utworzyć podpis e-mail, zobacz [Tworzenie i dodawanie podpisu do wiadomości w programie Outlook](https://support.office.com/article/8ee5d4f4-68fd-464a-a1c1-0e1c80bb27f2.aspx) lub Tworzenie i dodawanie [podpisu e-mail w aplikacji Outlook w sieci Web](https://support.office.com/article/5ff9dcfd-d3f1-447b-b2e9-39f91b074ea3.aspx).
