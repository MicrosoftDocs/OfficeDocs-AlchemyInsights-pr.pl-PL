# <a name="service-principal-key"></a>Klucz główny usługi

Aby uzyskać wskazówki dotyczące zarządzania poświadczeniami klucza dla podmiotu zabezpieczeń usługi, zobacz następujące artykuły:

- [Get-AzureADServicePrincipalKeyCredential](https://docs.microsoft.com/powershell/module/azuread/get-azureadserviceprincipalkeycredential) -Aby uzyskać poświadczenia klucza dla podmiotu zabezpieczeń usługi przy użyciu programu PowerShell.

- [New-AzureADServicePrincipalKeyCredential](https://docs.microsoft.com/powershell/module/azuread/new-azureadserviceprincipalkeycredential) -aby utworzyć nowe poświadczenia klucza dla podmiotu zabezpieczeń usługi przy użyciu programu PowerShell.

- [Remove-AzureADServicePrincipalKeyCredential](https://docs.microsoft.com/powershell/module/azuread/remove-azureadserviceprincipalkeycredential) -aby usunąć poświadczenia klucza z podmiotu zabezpieczeń usługi przy użyciu programu PowerShell.
