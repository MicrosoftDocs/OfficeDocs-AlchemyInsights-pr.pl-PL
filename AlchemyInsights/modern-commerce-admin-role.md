# <a name="modern-commerce-administrator-role"></a>Rola administratora nowoczesnego handlu

Ta rola jest automatycznie przypisywana do użytkowników, którzy dokonali zakupu, aby dać im ograniczony dostęp do portalu administracyjnego w celu zarządzania tym zakupem. Ci użytkownicy nie są administratorami i nie mają podwyższonych uprawnień.
