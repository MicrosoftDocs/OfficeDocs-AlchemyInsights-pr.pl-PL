# <a name="how-to-select-all-in-outlookcom"></a>Jak "Zaznacz wszystko" w Outlook.com

Możesz wybrać pozycję tylko wtedy, gdy wszystkie wiadomości są widoczne. Ma to zapobiec przypadkowym akcjom dotyczącym wiadomości, których nie widzisz.

**Ctrl + kliknięcie** powoduje wybranie wielu pojedynczych wiadomości.

**Kombinacja klawiszy Shift + kliknij** zaznacza wszystkie wiadomości między pierwszą a ostatnią wybraną wiadomością.
