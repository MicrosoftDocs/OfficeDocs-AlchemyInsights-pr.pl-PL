# <a name="using-custom-policy-in-intune"></a>Za pomocą niestandardowych zasad w Windows Intune

Windows Intune zawiera wiele wbudowane ustawienia dla urządzeń Android, iOS, Mac OS i Windows. Można również utworzyć niestandardowy profil, a następnie wdrożyć te ustawienia niestandardowe do urządzeń za pomocą Windows Intune. Profile niestandardowe są przeznaczone do dodać ustawienia urządzenia i funkcje, które nie są wbudowane usługi Intune.

[Niestandardowe ustawienia dla android w Windows Intune](https://docs.microsoft.com/intune/custom-settings-android)

[Niestandardowe ustawienia dla urządzenia Android Enterprise w Windows Intune](https://docs.microsoft.com/intune/custom-settings-android-for-work)

[Niestandardowe ustawienia dla urządzeń iOS w Windows Intune](https://docs.microsoft.com/intune/custom-settings-ios)

[Niestandardowe ustawienia dla urządzenia macOS w Windows Intune](https://docs.microsoft.com/intune/custom-settings-macos)

[Niestandardowe ustawienia dla urządzenia systemu Windows 10 w Windows Intune](https://docs.microsoft.com/intune/custom-settings-windows-10)

[Niestandardowe ustawienia dla Windows podrabianego dla urządzeń firmy w Windows Intune](https://docs.microsoft.com/intune/custom-settings-windows-holographic)
