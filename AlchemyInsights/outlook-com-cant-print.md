# <a name="unable-to-print"></a>Nie można wydrukować

Mamy świadomość problemu polegającego na tym, że niektórym użytkownikom nie można drukować podczas korzystania z przeglądarek Safari i Edge. Gdy pracujemy nad rozwiązaniem problemu, użyj innej przeglądarki. Przepraszamy za niedogodności.
