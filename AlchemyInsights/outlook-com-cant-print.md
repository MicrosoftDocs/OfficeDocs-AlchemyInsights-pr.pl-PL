# <a name="unable-to-print"></a>Nie można drukować

Wiemy o problemie, w którym niektórzy użytkownicy nie mogą drukować podczas korzystania z przeglądarek Safari lub Edge. Chociaż pracujemy nad poprawkami do tego, użyj innej przeglądarki. Przepraszamy za niedogodności.
