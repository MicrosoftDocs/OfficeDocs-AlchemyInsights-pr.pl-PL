# <a name="microsoft-365-language-settings"></a>Ustawienia języka usługi Microsoft 365

Określ, czy chcesz zmienić język wyświetlania i inne ustawienia dla użytkowników lub dla całej dzierżawy.

- W przypadku użytkowników zobacz [Zmienianie języka wyświetlania i strefy czasowej w usłudze Microsoft 365 dla firm dla wszystkich aplikacji](https://support.microsoft.com/office/6f238bff-5252-441e-b32b-655d5d85d15b).
- W przypadku dzierżawy zobacz [Ustawianie ustawień językowych i regionalnych dla usługi Microsoft 365](https://docs.microsoft.com/office365/troubleshoot/access-management/set-language-and-region).
