# <a name="how-to-delete-a-folder-in-outlook-on-the-web"></a>Jak usunąć folder w aplikacji Outlook w sieci Web

1. W okienku folderów kliknij prawym przyciskiem myszy folder lub podfolder, który chcesz usunąć.
2. Wybierz pozycję **Usuń folder**.
3. Wybierz **przycisk OK** , aby potwierdzić.

Aby dowiedzieć się więcej, zobacz [Praca z folderami wiadomości w aplikacji Outlook w sieci Web](https://support.office.com/article/ae0f10d6-54e7-4f29-acd3-78cdc3fdcb9f).
