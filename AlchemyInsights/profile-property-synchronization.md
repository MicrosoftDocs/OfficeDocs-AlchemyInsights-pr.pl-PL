# <a name="profile-property-synchronization"></a>Synchronizacja właściwości profilu

Nie jest możliwe do mapowania dodatkowe właściwości przy przywozie AD do synchronizacji z usługi Active Directory Azure aplikacji profilu użytkownika. AD Import jest ograniczony do wstępnie skonfigurowane właściwości, aby zagwarantować spójne wydajność zadanie czasomierza.
  
[Więcej informacji na temat właściwości profilu, które są synchronizowane](https://go.microsoft.com/fwlink/?linkid=875671)
