# <a name="profile-property-synchronization"></a>Synchronizacja właściwości profilu

Nie można zamapować dodatkowe właściwości dla importu AD do synchronizacji z usługi Azure Active Directory do aplikacji profilu użytkownika. Import AD jest ograniczony do wstępnie skonfigurowanego zestawu właściwości, aby zagwarantować stałą wydajność zadania czasomierza.
  
[Dowiedz się więcej o właściwościach profilu, które są synchronizowane](https://go.microsoft.com/fwlink/?linkid=875671)
