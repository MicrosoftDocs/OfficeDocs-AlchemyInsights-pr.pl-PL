# <a name="profile-property-synchronization"></a>Synchronizacja właściwości profilu

Nie można zmapować dodatkowych właściwości importu usługi AD na synchronizację z usługi Azure Active Directory z aplikacją profilu użytkownika. Importowanie reklam jest ograniczone do wstępnie skonfigurowanego zestawu właściwości w celu zapewnienia spójnej wydajności zadania czasomierza.
  
[Dowiedz się więcej na temat właściwości profilu, które są synchronizowane](https://go.microsoft.com/fwlink/?linkid=875671)
