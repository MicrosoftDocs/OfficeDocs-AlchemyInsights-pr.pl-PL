# <a name="profile-property-synchronization"></a>Synchronizacja właściwości profilu

Nie można mapować dodatkowych właściwości importu usługi AD do synchronizacji z usługi Azure Active Directory do aplikacji profilu użytkownika. Import usługi AD jest ograniczony do wstępnie skonfigurowanego zestawu właściwości, aby zagwarantować stałą wydajność zadania czasomierza.
  
[Dowiedz się więcej o synchronizacjach właściwości profilu](https://go.microsoft.com/fwlink/?linkid=875671)
