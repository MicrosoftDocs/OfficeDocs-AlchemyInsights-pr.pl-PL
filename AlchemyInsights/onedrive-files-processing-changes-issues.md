# <a name="onedrive-troubleshoot-processing-changes"></a>OneDrive: Rozwiązywanie problemów dotyczących przetwarzania zmian

Jeśli zostanie wyświetlony komunikat "przetwarzanie zmian" przez dłuższy czas, może to być spowodowane otwarciem pliku online, liczbą plików zablokowanych w kolejce lub bardzo dużym synchronizowaniem plików, zobacz [OneDrive: Rozwiązywanie problemów z przetwarzaniem zmian](https://support.office.com/article/onedrive-is-stuck-on-processing-changes-b386b813-9b66-4e47-8c4c-2b45533edccd) w celu uzyskania pomocy w rozwiązaniu tego problemu.

Ten komunikat zostanie wyświetlony również w przypadku, gdy plik zostanie zmieniony przez cały czas (taki jak program Outlook). Plik PST lub baza danych). Dowiedz się, jak [przenieść pliki danych programu Outlook](https://support.office.com/article/how-to-remove-an-outlook-pst-data-file-from-onedrive-b6b9e522-59bd-40f7-949f-168d0aa9b38e) , których używasz regularnie z usługi OneDrive.
