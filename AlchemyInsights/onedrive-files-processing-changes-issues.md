# <a name="onedrive-troubleshoot-processing-changes"></a>OneDrive: Rozwiązywanie problemów "przetwarzanie zmian"

Jeśli widzisz "przetwarzanie zmian" przez dłuższy czas, może to być, ponieważ masz otwarty plik online, wiele plików zatrzymany w kolejce lub bardzo dużych plików synchronizacji, zobacz [OneDrive zatrzymany na przetwarzanie](https://support.office.com/article/onedrive-is-stuck-on-processing-changes-b386b813-9b66-4e47-8c4c-2b45533edccd).

Przeczytaj ten artykuł, aby rozwiązać problem "przetwarzanie zmian". Ten komunikat będzie również wyświetlany, jeśli masz plik, który zmienia się cały czas (jak Outlook. Pliku PST lub bazy danych). Dowiedz się, jak [przenieść pliki danych programu Outlook](https://support.office.com/article/how-to-remove-an-outlook-pst-data-file-from-onedrive-b6b9e522-59bd-40f7-949f-168d0aa9b38e) , które są regularnie używane w usłudze OneDrive.
