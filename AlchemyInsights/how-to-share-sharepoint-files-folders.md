# <a name="how-to-share-sharepoint-files-or-folders"></a>Jak udostępniać pliki lub foldery programu SharePoint

Pliki przechowywane w witrynie programu SharePoint są zwykle dostępne dla wszystkich osób z uprawnieniami do witryny, ale możesz udostępnić określone pliki lub foldery osobom, które w przeciwnym razie nie mają dostępu do witryny. Podczas udostępniania plików i folderów można zdecydować, czy pozwolić ludziom edytować, czy po prostu je wyświetlać.

[Udostępnianie plików lub folderów programu SharePoint](https://support.office.com/article/1fe37332-0f9a-4719-970e-d2578da4941c)
