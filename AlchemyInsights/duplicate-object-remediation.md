# <a name="duplicate-object-synchronization-errors"></a>Zduplikowane błędy synchronizacji obiektów

Jeśli podczas synchronizacji zostanie wyświetlony co najmniej jedno z następujących błędów:

- Zsynchronizowany obiekt o tym samym adresie serwera proxy już istnieje w katalogu usługi Microsoft Online Services.

- Podobna wiadomość dotycząca elementu UserPrincipalName.

Użyj nowej diagnostyki kondycji AADConnect, aby wyeliminować zduplikowane błędy obiektów. Aby uzyskać więcej informacji, zobacz [diagnozowanie i korygowanie błędów synchronizacji zduplikowanych atrybutów](https://docs.microsoft.com/azure/active-directory/hybrid/how-to-connect-health-diagnose-sync-errors).
