# <a name="learn-about-site-compatibilityaffecting-changes-coming-to-microsoft-edge"></a>Informacje o zgodności witryny — wpływ na zmiany wprowadzone w programie Microsoft Edge

Sieć Web nieustannie rozwija się, aby usprawnić środowisko użytkownika, bezpieczeństwo i prywatność. Niektóre zmiany są na tyle znaczące, aby wpływać na funkcjonalność istniejących stron. Program Microsoft Edge Team aktualizuje strony w miarę rozwoju, osi czasu Solidify oraz [ogłasza zmiany](https://go.microsoft.com/fwlink/?linkid=2135534).
