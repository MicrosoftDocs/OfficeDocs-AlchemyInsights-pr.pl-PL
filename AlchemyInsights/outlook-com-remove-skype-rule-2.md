# <a name="how-do-i-turn-off-skype-in-outlookcom"></a>Jak wyłączyć program Skype w Hotmail?

Nie można usunąć programu Skype z Hotmail, ale można wyłączyć powiadomienia:

1. Otwórz okno **Ustawienia** > **Ogólne** > **[powiadomienia](https://outlook.live.com/mail/options/general/notifications)**. 
2. W obszarze **Czat i Rozmowa**Wyłącz typów powiadomień, które nie jest już potrzebne.

Aby uzyskać więcej informacji dotyczących korzystania z programu Skype przeczytaj [porozmawiać z rodziną i przyjaciółmi na Skype w Hotmail](https://support.office.com/article/83c6a5b1-3921-479c-b9e9-e753ce59c1fa).
