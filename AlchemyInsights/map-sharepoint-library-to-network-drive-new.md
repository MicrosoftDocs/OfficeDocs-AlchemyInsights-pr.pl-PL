# <a name="map-a-sharepoint-library-to-a-network-drive"></a>Mapuj na dysk sieciowy w bibliotece programu SharePoint

Mapowanie biblioteki jako dysk sieciowy jest tymczasowy i obsługiwane tylko przez program Internet Explorer. Od czasu do czasu muszą Otwórz witrynę programu SharePoint w programie Internet Explorer i wybierz "Zapamiętaj mnie", aby zapobiec upływającym sesji. Zamiast tego [Zsynchronizuj pliki programu SharePoint z nowego klienta synchronizacji OneDrive](https://support.office.com/article/sync-sharepoint-files-with-the-new-onedrive-sync-client-6de9ede8-5b6e-4503-80b2-6190f3354a88) </a> którym udostępnia [Pliki na żądanie](https://support.office.com/article/learn-about-onedrive-files-on-demand-0e6860d3-d9f3-4971-b321-7092438fb38e). Dostęp do wszystkich plików w OneDrive bez użycia lokalnej pamięci masowej.

Jeśli zdecydujesz się mapowanie dysku zamiast[przy użyciu nowego klienta synchronizacji OneDrive](https://support.office.com/article/sync-sharepoint-files-with-the-new-onedrive-sync-client-6de9ede8-5b6e-4503-80b2-6190f3354a88), upewnij się, że postępuj zgodnie z instrukcjami w artykule poniżej. 


**Jak skonfigurować i rozwiązywanie problemów z zmapowanych dysków sieciowych**


- [Konfigurowania i rozwiązywania problemów z zmapowanych dysków sieciowych](https://support.office.com/article/troubleshoot-mapped-network-drives-that-connect-to-sharepoint-online-ef399c67-4578-4c3a-adbe-0b489084eabe?ui=en-US&amp;rs=en-US&amp;ad=US).

Uwaga: Program Internet Explorer 10 z systemu Windows 8, Windows 7, które otrzymują "Odmowa dostępu" lub "Ścieżka jest niedostępna" podczas mapowania dysku, zainstalować tę poprawkę, aby rozwiązać ten problem. Przejdź do [błędu podczas otwierania biblioteki dokumentów programu SharePoint w programie Eksplorator Windows lub Mapuj dysk sieciowy do biblioteki, po zainstalowaniu programu Internet Explorer 10](https://support.microsoft.com/help/2846960).
