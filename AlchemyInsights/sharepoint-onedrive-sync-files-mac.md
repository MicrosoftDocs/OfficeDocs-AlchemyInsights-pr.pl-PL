# <a name="install-setup-and-sync-onedrive-or-sharepoint-files-on-mac"></a>Instalacji, konfiguracji i synchronizacji OneDrive lub pliki programu SharePoint w systemie Mac 

Zainstaluj Instalatora i synchronizacji plików OneDrive lub programu SharePoint na komputerze Mac z instrukcjami z [synchronizacji plików z klienta synchronizacji OneDrive w systemie Mac OS X](https://support.office.com/article/sync-files-with-the-onedrive-sync-client-on-mac-os-x-d11b9f29-00bb-4172-be39-997da46f913f).

Jeśli zainstalowano pogrubioną i instalacji na komputerze Mac, ale użytkownicy występują problemy z synchronizacją, zobacz [problemy synchronizacji poprawka OneDrive (Mac)](https://support.office.com/article/fix-onedrive-sync-problems-on-a-mac-af3012d7-13ec-4ac9-bbb1-ebcd2a0cd756?ui=en-US&amp;rs=en-US&amp;ad=US).

> [!NOTE]
> Funkcja znana przenieść Folder synchronizacji OneDrive dla systemu Windows nie jest dostępna dla komputerów Macintosh.
