# <a name="delete-a-classic-sharepoint-team-site-or-subsite"></a>Usuwanie klasycznych witryna zespołu programu SharePoint lub podwitryny

1. Przejdź do witryny.
    
2. Kliknij ikonę **Ustawienia** w prawym górnym rogu. 
    
3. Kliknij opcję **Ustawienia witryny**. W obszarze **Akcje witryny**kliknij przycisk **Usuń tę witrynę**, a następnie kliknij przycisk **Usuń**.
