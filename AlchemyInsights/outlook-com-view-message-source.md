# <a name="how-to-view-message-headers-or-message-source"></a>Sposób wyświetlania nagłówków wiadomości lub źródło wiadomości

Aby wyświetlić nagłówki i inne szczegóły w wiadomości e-mail, wybierz trzy kropki w górnym rogu okienka odczytu i wybierz **Widok źródła wiadomości**.
