# <a name="activating-windows-10-subscriptions"></a>Aktywowanie subskrypcji systemu Windows 10

Microsoft 365 Business Premium, Microsoft 365 Education, Microsoft 365 i Enterprise zapewniają licencje cyfrowe dla systemu Windows 10 w ramach subskrypcji. Aby można było aktywować subskrypcję, na urządzeniu musi być już aktywowany system Windows 10 Pro (1703 lub nowszy). Aby wyświetlić pełną listę wymagań dotyczących aktywacji subskrypcji systemu Windows 10, zobacz [Wymagania dotyczące aktywacji subskrypcji systemu Windows 10](https://docs.microsoft.com/windows/deployment/windows-10-subscription-activation#requirements).
