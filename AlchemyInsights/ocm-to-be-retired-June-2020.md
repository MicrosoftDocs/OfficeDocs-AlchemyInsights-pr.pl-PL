# <a name="ocm-to-be-retired-june-2020"></a>OCM zostanie wycofany w czerwcu 2020 r.

Po czerwcu 2020 r. nie będzie można korzystać z programu Outlook Customer Manager (OCM), ale będzie można wyeksportować lub usunąć dane. 

Aby uzyskać więcej informacji, zobacz:

- [Eksportowanie lub usuwanie danych programu Outlook Customer Manager](https://support.office.com/en-us/article/1a421cb4-e8de-4b44-bfb8-710b92820439)

- [Pytania dotyczące menedżera klienta programu Outlook](https://support.office.com/article/88e127ca-43a1-4c9d-8d52-6ad3a80f9c32)
