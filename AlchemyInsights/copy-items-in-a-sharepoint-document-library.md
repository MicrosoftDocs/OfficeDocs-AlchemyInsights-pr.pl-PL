# <a name="copy-files-in-sharepoint-and-onedrive"></a>Kopiowanie plików w programie SharePoint i usłudze OneDrive

Pliki z przeglądarki internetowej można kopiować do różnych lokalizacji w programie SharePoint i w usłudze OneDrive.

Aby uzyskać więcej informacji, zobacz [przenoszenie lub kopiowanie](https://support.microsoft.com/office/00e2f483-4df3-46be-a861-1f5f0c1a87bc)

> [!NOTE]
> **Kopiowanie** jest niedostępne, jeśli korzystasz z klasycznego środowiska usługi SharePoint Online.
