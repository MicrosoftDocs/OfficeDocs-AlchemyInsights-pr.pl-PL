# <a name="changing-shared-mailbox-send-as-behavior"></a>Zmiana zachowania udostępnionej skrzynki pocztowej Wyślij jako

Domyślnie wiadomości wysłane z udostępnionej skrzynki pocztowej nie są zapisywane w folderze Elementy wysłane tej skrzynki. Zamiast tego są zapisywane w folderach Elementy wysłane osób, które je wysłały.
  
Jeśli chcesz, aby wszyscy członkowie zobaczyli wysłana wiadomość e-mail, w centrum administracyjnym usługi Microsoft 365 edytuj szczegóły udostępnionej skrzynki pocztowej i wybierz pozycję **Wysłane elementy** \> **Edytuj**.
  
Aby dowiedzieć się więcej, zobacz [Tworzenie udostępnionej skrzynki pocztowej](https://docs.microsoft.com/microsoft-365/admin/email/create-a-shared-mailbox).
