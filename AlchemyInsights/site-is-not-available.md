# <a name="site-is-not-available"></a>Witryna jest niedostępna

Jeśli witryna programu SharePoint lub OneDrive nie jest dostępna dla wielu użytkowników, może to być tymczasowy problem z usługą. Sprawdź [pulpit nawigacyjny kondycji usługi,](https://admin.microsoft.com/AdminPortal/Home#/servicehealth) aby sprawdzić, czy dzierżawa ma wpływ.
