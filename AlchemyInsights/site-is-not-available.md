# <a name="site-is-not-available"></a>Strona nie jest dostępna

Jeśli witryna programu SharePoint lub OneDrive nie jest dostępna dla wielu użytkowników, może to być problem tymczasowy usługi. Sprawdź [pulpit nawigacyjny kondycji usługi](https://admin.microsoft.com/AdminPortal/Home#/servicehealth) , aby sprawdzić, czy Twoja dzierżawa ma wpływ.
