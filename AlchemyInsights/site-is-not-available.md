# <a name="site-is-not-available"></a>Strona nie jest dostępny

Jeśli witryny programu SharePoint lub OneDrive nie jest dostępny dla wielu użytkowników, może to być problem tymczasowy usługi. Sprawdź[Pulpit nawigacyjny kondycji usługi](https://admin.microsoft.com/AdminPortal/Home#/servicehealth) , by zobaczyć, jeśli dotyczy to dzierżawcy.
