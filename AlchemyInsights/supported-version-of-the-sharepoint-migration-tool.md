# <a name="supported-version-of-the-sharepoint-migration-tool"></a>Obsługiwana wersja narzędzia migracji programu SharePoint



Obecnie narzędzie migracji programu SharePoint obsługuje tylko migracja z programu SharePoint 2013. Wersji 2010 i 2016 nie są obsługiwane w tej chwili.
  
Aby uzyskać więcej informacji zobacz [Narzędzie migracji programu SharePoint](https://go.microsoft.com/fwlink/?linkid=2044765&amp;clcid=0x409).
