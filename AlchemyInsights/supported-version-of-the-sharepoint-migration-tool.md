# <a name="supported-version-of-the-sharepoint-migration-tool"></a>Obsługiwana wersja narzędzia migracji programu SharePoint



Obecnie narzędzie migracji programu SharePoint obsługuje tylko migracje z programu SharePoint 2013. Wersje 2010 i 2016 nie są obecnie obsługiwane.
  
Aby uzyskać więcej informacji, zobacz [Wprowadzenie narzędzia migracji programu SharePoint](https://go.microsoft.com/fwlink/?linkid=2044765&amp;clcid=0x409).
