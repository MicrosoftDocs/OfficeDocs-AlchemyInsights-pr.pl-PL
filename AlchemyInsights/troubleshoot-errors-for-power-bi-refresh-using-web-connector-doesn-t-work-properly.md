# <a name="refresh-using-web-connector-doesnt-work-properly"></a>Odświeżanie przy użyciu łącznika sieci Web nie działa poprawnie

Jeśli masz skrypt łącznika sieci Web, który używa funkcji [Web. Page](https://msdn.microsoft.com/library/mt260924.aspx) i zaktualizowaniu zestawu danych lub raportu po 18 listopada, 2016, należy użyć bramy w celu odświeżenia do poprawnego działania.

Aby uzyskać więcej informacji:[https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios](https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios)
