# <a name="refresh-using-web-connector-doesnt-work-properly"></a>Odśwież przy użyciu sieci Web connector nie działa poprawnie

Jeśli łącznik skrypt sieci web, która korzysta z funkcji [Web.Page](https://msdn.microsoft.com/library/mt260924.aspx) , i z zestawu danych lub raport był aktualizowany po 18 listopada 2016 należy użyć bramy w celu odświeżenia do poprawnego działania.

Aby uzyskać więcej informacji:[https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios](https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios)
