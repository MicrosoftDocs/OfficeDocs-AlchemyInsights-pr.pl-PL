# <a name="use-jamf-to-configure-microsoft-edge-policy-settings-on-macos"></a>Konfigurowanie ustawień zasad programu Microsoft Edge na macOS za pomocą Jamf

Aby skonfigurować program Microsoft Edge (stabilny kanał 81 lub nowszy) na macOS, użyj pliku manifestu zasad programu Microsoft Edge na Jamf Pro 10,19.

Aby uzyskać więcej informacji, zobacz [Konfigurowanie ustawień zasad programu Microsoft Edge dla macOS przy użyciu Jamf](https://go.microsoft.com/fwlink/?linkid=2134761).
