# <a name="use-your-browser-to-check-spelling"></a>Sprawdzanie pisowni za pomocą przeglądarki

Nie ma żadnych opcji wyboru pisowni w Hotmail. Aby sprawdzić pisownię, będziesz musiał użyć przeglądarki sieci web. Sprawdzanie pisowni jest dostępny w Microsoft Edge, Internet Explorer 10 i nowszych wersjach i aktualne wersje przeglądarki Firefox, Chrome i Safari. Sprawdź opcje przeglądarki sieci web dowiedzieć się więcej o sprawdzaniu pisowni.
