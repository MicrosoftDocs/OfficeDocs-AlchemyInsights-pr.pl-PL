# <a name="internal-server-errors-in-outlookcom"></a>Wewnętrzne błędy serwera w Outlook.com

O nie! Jeśli widzisz komunikat "Error 500" lub "Internal Server Error", to my, a nie ty - przepraszam.

Sprawdź [stronę Stan usługi pakietu Office,](https://portal.office.com/servicestatus) aby upewnić się, że nic się nie dzieje, ale najlepszym rozwiązaniem jest zrobienie przerwy na kilka minut, a następnie odświeżenie przeglądarki, naciskając **klawisze CTRL** i **F5**.
