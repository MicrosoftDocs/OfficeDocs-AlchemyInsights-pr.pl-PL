# <a name="identifiers-entity-id"></a>Identyfikatory (identyfikator jednostki)

Aby wyświetlić lub edytować oświadczenia wystawione w tokenie SAML aplikacji, Otwórz aplikację w portalu Azure Portal. Następnie otwórz sekcję " **atrybuty użytkownika & oświadczenia** ". Aby uzyskać więcej informacji, zobacz [Dostosowywanie oświadczeń wydanych w tokenie SAML dla aplikacji dla przedsiębiorstw](https://docs.microsoft.com/azure/active-directory/develop/active-directory-saml-claims-customization#editing-nameid).
