# <a name="cant-delete-files-or-photos-from-files-in-outlook-on-the-web"></a>Nie można usunąć plików ani zdjęć z plików w aplikacji Outlook w sieci Web

**Pliki** pokazują pliki i zdjęcia dołączone do wiadomości w skrzynce pocztowej. Aby usunąć pliki i zdjęcia z plików, musisz usunąć dołączone wiadomości e-mail. Po usunięciu wiadomości należy również usunąć je z folderu Elementy usunięte.
