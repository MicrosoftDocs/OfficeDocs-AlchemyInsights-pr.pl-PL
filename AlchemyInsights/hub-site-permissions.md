# <a name="hub-site-permissions"></a>Uprawnienia witryny centrum

Zarządzanie uprawnieniami witryny centrum zależy od typu witryny. Jeśli witryna należy do witryny zespołu połączonej z grupą, należy zarządzać uprawnieniami za pośrednictwem grupy Microsoft 365. Jeśli jest to witryna komunikacyjna, należy zarządzać uprawnieniami za pośrednictwem grup programu SharePoint.

- [Uprawnienia witryny centrum](https://docs.microsoft.com/sharepoint/modern-experience-sharing-permissions#hub-site-permissions)  
- [Planowanie witryn centrum](https://docs.microsoft.com/sharepoint/planning-hub-sites)
