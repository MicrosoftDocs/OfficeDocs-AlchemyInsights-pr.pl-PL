# <a name="550-57705-access-denied-tenant-has-exceeded-threshold"></a>550 5.7.705 Odmowa dostępu, dzierżawca przekroczyła próg

Ta wiadomość pojawia się, gdy organizacja wysłała zbyt dużo spamu lub poczty zbiorczej, a my umieszczamy blok na poczcie wychodzącej.
Typowe przyczyny są zagrożone na serwerach lokalnych lub przejęte konta administratora, które zostały użyte do tworzenia łączników. Każdy warunek może pozwolić na przekazywanie spamu przez twoją organizację.

Aby usunąć ten blok, musisz zrozumieć i wyjaśnić przyczynę agentowi pomocy technicznej, a także rozwiązać podstawowy problem.
Kliknij **skontaktuj się z nami** poniżej, gdy będziesz gotowy.
