# <a name="diagnose-and-resolve-outlook-connectivity-issues"></a>Diagnozowanie i rozwiązywanie problemów z łącznością programu Outlook

Aby zdiagnozować i rozwiązać problemy z łącznością programu Outlook, można uruchomić [narzędzie łączności programu Outlook](https://aka.ms/SaRA-OutlookDisconnect-Alchemy) na komputerze, którego dotyczy problem.
  
To narzędzie przeprowadzi testy diagnostyczne i zasugeruje możliwe rozwiązania problemu z łącznością programu Outlook.
