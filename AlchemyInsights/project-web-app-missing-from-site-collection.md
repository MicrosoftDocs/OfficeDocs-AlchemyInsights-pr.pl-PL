# <a name="project-web-app-is-missing-from-the-sharepoint-site-collection"></a>W zbiorze witryn programu SharePoint brakuje aplikacji Project Web App

Z następujących powodów w zbiorze witryn programu SharePoint może brakować aplikacji Project Web App:

1. Organizacje mają tylko licencje (licencje) programu Project Online Essentials. Administratorzy będą musieli dodać licencję Project Online Professional i/lub Premium. Strona wyjdzie z tylko do odczytu w tym momencie. Aby uzyskać więcej informacji, zobacz [Porównanie rozwiązań do zarządzania projektami](https://products.office.com/project/compare-microsoft-project-management-software?tab=1).
2. Dzierżawca napotkał problem inicjowania obsługi administracyjnej, w którym inicjowanie obsługi administracyjnej aplikacji aplikacji Project Web App została opóźniona. Jeśli minęło więcej niż 24 godziny i nadal występuje problem, zaloguj się na bilet pomocy technicznej. Rozumiemy, że 24 godziny mogą wydawać się długie. W wielu przypadkach pracujemy już nad rozwiązaniem. Proszę dać nam co najmniej 24 godziny na uzupełnienie rozwiązania.
