# <a name="managing-microsoft-stream-licenses"></a>Zarządzanie licencjami Microsoft Stream

**Ważne:** Aby wykonać operacje opisane w tym temacie, musisz być w roli administratora dzierżawy.

Musisz być administratorem globalnym lub administratorem zarządzania użytkownikami. Aby uzyskać więcej informacji, zobacz [role administratora — informacje](https://docs.microsoft.com/microsoft-365/admin/add-users/about-admin-roles).

Możesz przypisywać [licencje do kont użytkowników przy użyciu programu Office 365 PowerShell](https://go.microsoft.com/fwlink/p/?linkid=850410).
