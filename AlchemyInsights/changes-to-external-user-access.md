# <a name="changes-to-external-user-access"></a>Zmiany do dostępu użytkowników zewnętrznych

Od 23 marca 2018, użytkownicy zewnętrzni nie będą mogli uzyskać dostęp do zawartości, który został udostępniony wszystkim, wszyscy uwierzytelnieni użytkownicy lub wszyscy użytkownicy formularzy, nawet jeśli zostały zaproszone lub dodane do grupy autoryzowane. Użytkownicy zewnętrzni będą mogli uzyskać dostęp do zawartości, który został udostępniony z użytkownikiem lub z grup, do których należy użytkownik. Jeśli chcesz zachować istniejące ustawienia, zobacz:https://support.microsoft.com/help/4089534/how-to-grant-the-everyone-claim-to-external-users-in-office-365
