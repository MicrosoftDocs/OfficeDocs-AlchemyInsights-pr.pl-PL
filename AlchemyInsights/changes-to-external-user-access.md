# <a name="changes-to-external-user-access"></a>Zmiany w dostępie użytkowników zewnętrznych

Począwszy od 23 marca 2018, użytkownicy zewnętrzni nie będą już mogli uzyskiwać dostępu do zawartości udostępnionej wszystkim użytkownikom, wszystkim użytkownikom uwierzytelnionym lub wszyscy użytkownicy formularzy, nawet jeśli zostali zaproszeni lub dodani do autoryzowanych grup. Użytkownicy zewnętrzni będą mogli uzyskiwać dostęp tylko do zawartości udostępnionej użytkownikowi lub grupom, do których należy użytkownik. Jeśli chcesz zachować istniejące ustawienia, zapoznaj się z:https://support.microsoft.com/help/4089534/how-to-grant-the-everyone-claim-to-external-users-in-office-365
