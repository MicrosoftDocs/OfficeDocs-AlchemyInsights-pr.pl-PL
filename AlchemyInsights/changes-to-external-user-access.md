# <a name="changes-to-external-user-access"></a>Zmiany w dostępie użytkowników zewnętrznych

Od 23 marca 2018 r. użytkownicy zewnętrzni nie będą już mogli uzyskiwać dostępu do zawartości udostępnionej wszystkim użytkownikom, wszystkim uwierzytelnionym użytkownikom lub wszystkim użytkownikom formularzy, nawet jeśli zostali zaproszeni lub dodani do autoryzowanych grup. Użytkownicy zewnętrzni będą mogli uzyskiwać dostęp tylko do zawartości udostępnionej użytkownikowi lub grupom, do których należy użytkownik. Jeśli chcesz zachować istniejące ustawienia, zobacz:https://support.microsoft.com/help/4089534/how-to-grant-the-everyone-claim-to-external-users-in-office-365
