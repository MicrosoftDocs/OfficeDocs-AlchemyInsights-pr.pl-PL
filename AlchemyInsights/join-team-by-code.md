# <a name="join-a-team-by-code"></a>Dołączanie do zespołu za pomocą kodu

Firma Microsoft wie o problemie,  który dotyczy użytkowników, którzy nie mogą utworzyć kodu dołączania do zespołu lub dołączania do zespołu przy użyciu funkcji Dołącz **za pomocą** kodu. Aby obejść ten problem, właściciele zespołu mogą ręcznie dodawać użytkowników, w razie potrzeby.
