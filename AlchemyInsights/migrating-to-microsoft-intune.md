# <a name="migrating-to-microsoft-intune"></a>Migracja do usługi Microsoft Intune

Jeśli próbujesz przeprowadzić migrację do usługi Intune, przejrzyj [przewodnik po migracji usługi Intune](https://docs.microsoft.com/intune/fundamentals/migration-guide).

Aby uzyskać bardziej szczegółowe wskazówki dotyczące migracji z programu System Center Configuration Manager (i SCCM Hybrid) do usługi Intune, należy ponownie przygotować [dokumentację migracji hybrydowej.](https://docs.microsoft.com/sccm/mdm/deploy-use/migrate-hybridmdm-to-intunesa)
