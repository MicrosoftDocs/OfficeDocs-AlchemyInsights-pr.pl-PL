# <a name="migrating-to-microsoft-intune"></a>Migrowanie do usługi Microsoft Intune

Jeśli próbujesz przeprowadzić migrację do usługi Intune, przejrzyj [Przewodnik po migracji usługi Intune](https://docs.microsoft.com/intune/fundamentals/migration-guide).

Aby uzyskać bardziej szczegółowe wskazówki dotyczące migrowania z programu System Center Configuration Manager (i hybrydowego programu SCCM) do usługi Intune, Reivew naszej [dokumentacji migracji hybrydowej](https://docs.microsoft.com/sccm/mdm/deploy-use/migrate-hybridmdm-to-intunesa).
