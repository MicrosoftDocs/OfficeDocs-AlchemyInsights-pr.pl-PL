# <a name="removing-the-last-on-premises-exchange-server"></a>Usuwanie ostatniego lokalnego programu Exchange Server

Przed usunięciem ostatniego serwera programu Exchange z organizacji lokalnej, należy się upewnić, że rozumiesz, jakie są implikacje tej czynności, aby ułatwić podjęcie decyzji, czy potrzebny jest ostatni serwer do innych celów. Aby uzyskać więcej informacji, zapoznaj się z informacjami o [tym, jak i kiedy zlikwidować lokalne serwery programu Exchange we wdrożeniu hybrydowym](https://technet.microsoft.com/library/dn931280%28v=exchg.150%29.aspx).
