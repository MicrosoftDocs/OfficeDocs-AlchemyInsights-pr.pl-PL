# <a name="removing-the-last-on-premises-exchange-server"></a>Usunięcie ostatniego na lokalnym serwerze Exchange

Przed usunięciem ostatniego serwera programu Exchange z organizacji lokalnej, należy upewnij się, że zrozumieć implikacje robi to, aby pomóc zdecydować, jeśli potrzebujesz tego ostatniego serwera do innych celów. Aby uzyskać więcej informacji, zapoznaj się [jak i kiedy należy zlikwidować lokalnych serwerów Exchange w danym wdrożeniu hybrydowy](https://technet.microsoft.com/library/dn931280%28v=exchg.150%29.aspx).
