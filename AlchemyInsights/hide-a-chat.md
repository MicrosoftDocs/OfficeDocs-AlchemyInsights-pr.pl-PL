# <a name="hide-a-chat"></a>Ukrywanie czatu

Nie możesz usunąć całego czatu (lub konwersacji), ale możesz ją ukryć. Możesz również usunąć wysłaną [wiadomość](https://support.office.com/client/delete-a-message-you-have-sent-67bd76a5-04e7-46ea-9ef0-5800865cb8f3) .

Aby ukryć czat:

- Wybierz pozycję **czat** z menu po lewej stronie.
- Wybierz czat, który chcesz ukryć.
- Wybierz trzy kropki (**...**) obok czatu, a następnie wybierz pozycję **Ukryj**.
