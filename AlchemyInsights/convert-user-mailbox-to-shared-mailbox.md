# <a name="convert-a-user-mailbox-to-a-room-or-equipment-mailbox"></a>Konwertowanie skrzynki pocztowej użytkownika na pomieszczenie lub sprzęt skrzynki pocztowej

Za pomocą środowiska PowerShell Online programu Exchange, można przekonwertować skrzynki pocztowej użytkownika do skrzynki pocztowej pomieszczenie lub sprzęt. Zobacz [Konwertowanie skrzynki pocztowej](https://go.microsoft.com/fwlink/p/?LinkId=832875) , aby uzyskać więcej szczegółów.
