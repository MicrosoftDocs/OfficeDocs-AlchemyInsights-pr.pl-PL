# <a name="use-sara-to-diagnose-and-resolve-outlook-authentication-issues"></a>Użyj Ewa, aby zdiagnozować i rozwiązać problemy z uwierzytelnianiem programu Outlook

Zdecydowanie zaleca się że używasz [programu Outlook ciągle wywołuje pytanie o hasło](https://aka.ms/SaRA-OutlookPwdPrompt) diagnostycznych na komputer do rozwiązywania problemów z gdzie program Outlook nieustannie monituje o hasło. To [Ewa](https://diagnostics.office.com/#/) diagnostyczne wykonuje automatyczne sprawdzanie i zwraca możliwych rozwiązań do użycia do usunięcia wszelkich wykrytych luk.
