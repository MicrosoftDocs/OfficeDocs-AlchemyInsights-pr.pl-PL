# <a name="how-to-type-special-characters-on-a-mac"></a>Jak wpisać znaki specjalne na komputerach Mac

Aby wpisać znaki specjalne na komputerze Mac, otwórz menu **znaki specjalne** z skrót klawiaturowy ⌘ **polecenia** + **kontroli** + **miejsca**.

Szybki skrót dla @ jest również **ALT** + **64**.
