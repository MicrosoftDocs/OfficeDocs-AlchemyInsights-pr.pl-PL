# <a name="edit-a-read-only-file"></a>Edytowanie pliku tylko do odczytu

W bibliotekach dokumentów po kliknięciu na nazwę dokumentu, program pakietu Office może nieoczekiwanie go otworzyć jako plik tylko do odczytu. Aby otworzyć ją do edycji, kliknij przycisk **Otwórz** (trzy kropki), a następnie kliknij **Edytuj.**
