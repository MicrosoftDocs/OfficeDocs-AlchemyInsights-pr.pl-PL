# <a name="limit-sharing-in-microsoft-365"></a>Ogranicz udostępnianie w programie Microsoft 365

Chociaż nie możesz całkowicie wyłączyć udostępniania wewnętrznego lub usunąć przycisku Udostępnij z witryn, istnieje wiele sposobów ograniczenia współużytkowania w programie Microsoft 365 w celu realizacji potrzeb organizacji. Aby uzyskać więcej informacji, zobacz [ograniczanie udostępniania w programie Microsoft 365](https://docs.microsoft.com/Office365/Enterprise/microsoft-365-limit-sharing).
