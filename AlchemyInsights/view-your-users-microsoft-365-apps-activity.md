# <a name="view-your-users-microsoft-365-apps-activity"></a>Wyświetlanie aktywności aplikacji Microsoft 365 dla użytkowników

[Raport aktywacja pakietu Office](https://docs.microsoft.com/microsoft-365/admin/activity-reports/microsoft-office-activations?view=o365-worldwide) zawiera widok, w którym użytkownicy zainstalowali i aktywowali aplikacje Microsoft 365 na co najmniej jednym urządzeniu.

Ponadto możesz uzyskać widok aktywności aplikacji Microsoft 365 dla użytkowników przy użyciu [raportu Użycie aplikacji microsoft 365 Apps](https://docs.microsoft.com/microsoft-365/admin/activity-reports/microsoft365-apps-usage?view=o365-worldwide).
