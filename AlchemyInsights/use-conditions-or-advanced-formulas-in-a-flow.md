# <a name="use-conditions-or-advanced-formulas-in-a-flow"></a>Stosowanie warunków lub zaawansowanych formuł w przepływie

W programie Microsoft Dataflow można dostosować przepływ, tak aby wykonywał zadania tylko wtedy, gdy warunek jest prawdziwy. Aby uzyskać informacje, zobacz [Dodawanie warunku do przepływu](https://go.microsoft.com/fwlink/?linkid=872112).
  
Aby utworzyć formułę zaawansowaną, zobacz [schemat języka definicji przepływu pracy dla aplikacji logiki platformy Azure](https://aka.ms/logicexpressions).
