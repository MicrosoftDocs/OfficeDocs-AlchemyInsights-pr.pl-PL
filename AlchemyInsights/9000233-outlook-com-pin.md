# <a name="change-your-pin"></a>Zmień numer PIN

Hotmail nie korzystają z numeru PIN, ale mogą logować do komputera PC za pomocą numeru PIN lub Windows Hello. Aby zmienić numer PIN w systemie Windows 10:

1. Przejdź do menu Start, a następnie wybierz **Ustawienia** > **konta** > **Opcje logowania**.
2. Wybierz przycisk **Zmień** pod **numer PIN**.
3. Wprowadź bieżący numer PIN; następnie wprowadź i Potwierdź nowy numer PIN pod spodem.

[Dowiedz się więcej o funkcji Windows Hello.](https://support.microsoft.com/help/17215/)
