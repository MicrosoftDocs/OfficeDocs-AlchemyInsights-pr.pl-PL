# <a name="unable-to-add-an-attachment-to-an-externally-facing-form"></a>Nie można dodać załącznika do formularza zewnętrznego

Jeśli próbujesz dodać załącznik do zewnętrznej ankiety programu Microsoft Forms, ta funkcja nie jest jeszcze dostępna. Aby śledzić stan tego żądania funkcji, przejdź do [tego wpisu](https://go.microsoft.com/fwlink/?linkid=2133069)w usłudze UserVoice i Dodaj swój głos.
