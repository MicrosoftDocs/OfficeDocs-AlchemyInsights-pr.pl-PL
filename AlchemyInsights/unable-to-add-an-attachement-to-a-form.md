# <a name="unable-to-add-an-attachment-to-an-externally-facing-form"></a>Nie można dodać załącznika do formularza skierowanego z zewnątrz

Jeśli próbujesz dodać załącznik do zewnętrznej ankiety microsoft forms, ta funkcja nie jest jeszcze dostępna. Aby śledzić stan tego żądania funkcji, przejdź do [tego wpisu UserVoice](https://go.microsoft.com/fwlink/?linkid=2133069)i dodaj swój głos.
