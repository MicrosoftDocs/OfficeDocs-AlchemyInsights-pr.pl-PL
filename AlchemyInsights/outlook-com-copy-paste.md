# <a name="copy-and-paste-in-outlookcom"></a>Kopiowanie i wklejanie w Hotmail

Nie Kopiuj lub Wklej przyciski w Hotmail, ale można użyć skrótów klawiaturowych:

**Na komputerze PC:** CTRL + C, aby skopiować, CTRL + X, aby wyciąć i CTRL + V, aby wkleić.

**Na komputerze Mac:** CMD + C, aby skopiować, CMD + X, aby wyciąć i CMD + V, aby wkleić.
