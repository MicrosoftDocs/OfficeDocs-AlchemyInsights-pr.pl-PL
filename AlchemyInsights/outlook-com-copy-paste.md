# <a name="copy-and-paste-in-outlookcom"></a>Kopiowanie i wklejanie w Outlook.com

W Outlook.com nie ma przycisków kopiowania ani wklejanych, ale można używać skrótów klawiaturowych:

**Na komputerze:** CTRL+C do skopiowania, CTRL+X do wycięcia i CTRL+V do wklejenia.

**Na komputerze Mac:** CMD + C do kopiowania, CMD + X do cięcia i CMD + V do wklejenia.
