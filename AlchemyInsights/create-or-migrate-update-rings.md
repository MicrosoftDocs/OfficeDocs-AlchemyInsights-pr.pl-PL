# <a name="create-or-migrate-update-rings"></a>Tworzenie lub migrowanie pierścieni aktualizacji

Jeśli skonfigurowano ustawienia aktualizacji systemu Windows 10 w klasycznym portalu usługi Intune i chcesz przeprowadzić migrację ustawień do usługi Intune w witrynie Azure portal, wykonaj następujące kroki:

1.  Przejdź do witryny Azure portal i wybierz **pozycję Wszystkie usługi**.
2.  W polu **Filtr** wpisz pozycję **Intune**i wybierz pozycję **Microsoft Intune**.
3.  Wybierz **pozycję Aktualizacje oprogramowania**   >   **Tworzenie pierścieni aktualizacji systemu Windows 10**   >   **.**
4.  Wprowadź nazwę i opis, a następnie wybierz pozycję **Konfiguruj**.
5.  Skonfiguruj ustawienia aktualizacji oprogramowania dla swojej organizacji.
6.  Wybierz **pozycję**  >  **Utwórz pierścień aktualizacji**  >  **.**
