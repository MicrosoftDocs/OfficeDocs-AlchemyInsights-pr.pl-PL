# <a name="seeing-a-false-alert"></a>Wyświetlanie fałszywego alertu

- Prześlij plik do firmy Microsoft do analizy. Aby uzyskać więcej informacji, zobacz [Co zrobić z fałszywymi alarmami/negatywami w programie Microsoft Defender Antivirus](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-antivirus/antivirus-false-positives-negatives#submit-a-file-to-microsoft-for-analysis).
- Utwórz wskaźnik "Zezwól", aby zapobiec fałszywie dodatniemu wpisowi. Aby uzyskać więcej informacji, zobacz [Tworzenie wskaźników dla plików](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/indicator-file).  
- Skonfiguruj wykluczenia na urządzeniu, aby zapobiec skanowaniu elementu. Aby uzyskać więcej informacji, zobacz [Definiowanie wykluczenia na poszczególnych urządzeniach z systemem Windows, aby zapobiec skanowaniu elementu](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-antivirus/antivirus-false-positives-negatives#define-an-exclusion-on-an-individual-windows-device-to-prevent-an-item-from-being-scanned).
