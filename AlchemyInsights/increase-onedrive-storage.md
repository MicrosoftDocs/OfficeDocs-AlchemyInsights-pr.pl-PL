# <a name="increase-onedrive-storage"></a>Zwiększanie miejsca do magazynowania w usłudze OneDrive

Jako Administrator globalny lub administrator programu SharePoint w programie Microsoft 365 możesz ustawić miejsce do magazynowania w usłudze OneDrive dla określonego użytkownika bezpośrednio z centrum administracyjnego Microsoft.  

Aby uzyskać więcej informacji, zobacz [Zmienianie miejsca do magazynowania w usłudze OneDrive określonego użytkownika](https://docs.microsoft.com/onedrive/change-user-storage).
