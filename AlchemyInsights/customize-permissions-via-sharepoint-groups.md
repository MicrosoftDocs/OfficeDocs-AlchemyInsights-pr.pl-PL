# <a name="how-to-customize-sharepoint-site-permissions-via-sharepoint-groups"></a>Jak dostosować uprawnienia witryny programu SharePoint za pomocą grup programu SharePoint 

Grupa programu SharePoint to zbiór użytkowników, którzy mają ten sam zestaw uprawnień do witryn i zawartości. Zamiast przypisywać uprawnienia jednej osobie, można użyć grup w celu wygodnego przydzielenia tego samego poziomu uprawnień wielu osobom naraz.

[Dostosowywanie uprawnień do witryny programu SharePoint](https://docs.microsoft.com/sharepoint/customize-sharepoint-site-permissions)
