# <a name="block-file-download-with-azure-ad-conditional-access"></a>Blokowanie pobierania plików z dostępem warunkowym usługi Azure AD

Zasady sesji usługi Cloud App Security umożliwiają ograniczenie sesji na podstawie stanu urządzenia. Aby uzyskać kontrolę nad sesją przy użyciu urządzenia jako warunku, należy utworzyć zarówno zasady dostępu warunkowego, jak i zasady sesji.

- [Blokowanie pobierania plików z dostępem warunkowym usługi Azure AD](https://docs.microsoft.com/cloud-app-security/use-case-proxy-block-session-aad#create-a-block-download-policy-for-unmanaged-devices)
