# <a name="getting-an-account-disabled-error-in-outlook-on-the-web"></a>Wystąpił błąd konta wyłączony w aplikacji Outlook w sieci Web?

Jeśli podczas próby otwarcia programu Outlook `X-OWA-Error: Microsoft.Exchange.Data.Storage.AccountDisabledException` Web App (dawniej znanej jako Aplikacja Outlook Web App) zostanie wyświetlony błąd, z którym **konto zostało wyłączone,** administrator organizacji mógł wyłączyć dostęp do programu Outlook w sieci Web. Administrator może naprawić ten błąd, wykonując kroki opisane w temacie [Włączanie lub wyłączanie aplikacji Outlook Web App dla skrzynki pocztowej](https://technet.microsoft.com/library/bb124124%28v=exchg.150%29.aspx).
