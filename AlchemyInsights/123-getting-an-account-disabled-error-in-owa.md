# <a name="getting-an-account-disabled-error-in-outlook-on-the-web"></a>Uzyskiwanie konta wyłączone błąd w programie Outlook w sieci web?

Jeśli pojawi się błąd, **Twoje konto zostało wyłączone** z `X-OWA-Error: Microsoft.Exchange.Data.Storage.AccountDisabledException` podczas próby otwarcia programu Outlook w sieci web (dawniej Outlook Web App), Twój administrator organizacji mógł wyłączyć dostępu do programu Outlook w sieci web. Administrator może rozwiązać ten problem, wykonując kroki opisane w temacie [Włączanie lub wyłączanie aplikacji sieci Web programu Outlook dla skrzynki pocztowej](https://technet.microsoft.com/library/bb124124%28v=exchg.150%29.aspx).
