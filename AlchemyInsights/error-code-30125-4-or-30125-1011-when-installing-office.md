# <a name="solutions-for-error-30125-4-or-30125-1011"></a>Rozwiązania dla błędu 30125-4 lub 30125 1011

Ten błąd może wystąpić, jeśli program antywirusowy, zapora, ustawienia serwera proxy lub połączenie uniemożliwiają instalację pakietu Office. Oto kilka rzeczy, możesz spróbować rozwiązać problem:
  
- Uruchom ponownie komputer i ponownie zainstalować pakiet Office
    
- Zainstaluj pakiet Office przy użyciu [Instalatora pakietu Office w trybie offline](https://support.office.com/article/f0a85fe7-118f-41cb-a791-d59cef96ad1c.aspx)
    
- Tymczasowo wyłącz ustawienia serwera proxy
    
- Tymczasowo wyłącz oprogramowanie antywirusowe
    
- Tymczasowo wyłącz zaporę
    
Aby uzyskać więcej informacji, zobacz [Kod błędu 30125-4 lub 30125-1011 podczas instalowania pakietu Office](https://support.office.com/article/7bfabec6-76be-4cde-880e-819a9c569612.aspx)
