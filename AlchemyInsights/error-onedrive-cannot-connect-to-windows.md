# <a name="error-onedrive-cannot-connect-to-windows"></a>Błąd: Usługa OneDrive nie może nawiązać połączenia z systemem Windows

Aby rozpoznać komunikat o błędzie, Wyszukaj plik cldflt.sys, aby sprawdzić, czy jest uruchomiony, a jeśli nie, włącz go. 
  
Aby uzyskać dodatkowe informacje na temat włączania pliku cldflt.sys, zobacz [Usługa OneDrive nie może nawiązać połączenia z systemem Windows podczas uzyskiwania dostępu do plików w usłudze OneDrive.](https://go.microsoft.com/fwlink/?Linkid=2031032)
