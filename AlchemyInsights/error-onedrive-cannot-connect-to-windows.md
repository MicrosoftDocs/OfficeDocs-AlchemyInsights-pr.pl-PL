# <a name="error-onedrive-cannot-connect-to-windows"></a>Błąd: usługa OneDrive nie może połączyć się z systemem Windows

Aby rozwiązać ten komunikat o błędzie, wyszukaj plik cldflt.sys, aby sprawdzić, czy jest uruchomiony, a jeśli nie, włącz go. 
  
Aby uzyskać dodatkowe informacje dotyczące włączania pliku cldflt.sys, zobacz [Błąd usługi OneDrive nie może połączyć się z systemem Windows podczas uzyskiwania dostępu do plików w usłudze OneDrive.](https://go.microsoft.com/fwlink/?Linkid=2031032)
