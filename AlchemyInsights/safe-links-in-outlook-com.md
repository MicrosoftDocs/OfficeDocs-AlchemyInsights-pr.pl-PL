# <a name="disabling-advanced-outlookcom-security"></a>Wyłączenie zaawansowanych zabezpieczeń Hotmail

Aby zapewnić najlepszą ochronę dla swojego konta, Hotmail sprawdza, czy wszystkie łącza w wiadomościach phishingowych są lub mogą pobrać wirusy lub złośliwe oprogramowanie z komputera. Jeśli jesteś pewien, że chcesz usunięcie tych zaawansowanych zabezpieczeń Hotmail, kliknij poniżej przycisk **Tak** .
