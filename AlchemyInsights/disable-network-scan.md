# <a name="disable-network-scan"></a>Wyłączanie skanowania sieci

Skany udziału sieciowego mogą mieć wpływ na wydajność.  Aby upewnić się, że klient domyślnie nie skanuje udziałów/plików sieciowych, skonfiguruj następujące ustawienia w aplikacji Windows Defender na **Wartość Prawda:**

- Disable JednakoweDaneMappedNetworkDrivesForFull Jednakowe
- Disable JednakoweNetworkFiles
