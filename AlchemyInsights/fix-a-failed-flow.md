# <a name="fix-a-flow-that-failed"></a>Naprawianie przepływu, który uległ awarii

1. Przejdź do [witryny Microsoft Flow](https://flow.microsoft.com/)i kliknij ikonę powiadomienia w prawym górnym rogu. (W aplikacji mobilnej wybierz zakładkę aktywność). Wybierz przepływ, którego nie można sprawdzić szczegóły.
    
2. Znajdź krok z czerwoną ikoną wykrzyknik, aby wyświetlić komunikat o błędzie dla przepływu.
    
3. W zależności od komunikatu o błędzie powinieneś być w stanie edytować przepływ i naprawić go. 
    
[Typowe awarie przepływu](https://go.microsoft.com/fwlink/?linkid=872110)
