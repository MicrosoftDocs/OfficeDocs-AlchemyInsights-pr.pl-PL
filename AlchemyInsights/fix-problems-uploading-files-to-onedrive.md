# <a name="fix-problems-uploading-files-to-onedrive"></a>Rozwiąż problemy z przekazywaniem plików do OneDrive

Należy zastosować, jeśli nie można przesłać pliku:
  
- Zmień nazwę pliku i spróbuj ponownie. Niektóre nazwy plików i znaków w nazwach plików nie jest dozwolone. 
    
- Jeśli synchronizujesz plik, przenieść go z folderu OneDrive (na przykład przesunąć go na pulpicie), a kiedy OneDrive stan synchronizacji jest wyświetlany jako "Na bieżąco," przeniesienie go z powrotem.
