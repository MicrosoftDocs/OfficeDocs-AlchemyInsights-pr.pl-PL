# <a name="fix-problems-uploading-files-to-onedrive"></a>Rozwiązywanie problemów z przekazywaniem plików do usługi OneDrive

Spróbuj to zrobić, jeśli plik nie zostanie przekazany:
  
- Zmień nazwę pliku i spróbuj ponownie. Niektóre nazwy plików i znaki w nazwach plików są niedozwolone. 
    
- Jeśli synchronizujesz plik, przenieś go z folderu usługi OneDrive (na przykład przenieś go na pulpit), a gdy stan synchronizacji usługi OneDrive pojawi się jako "Aktualna", przenieś go z powrotem.
