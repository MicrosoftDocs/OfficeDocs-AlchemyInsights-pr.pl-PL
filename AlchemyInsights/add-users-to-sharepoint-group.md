# <a name="add-users-to-a-sharepoint-group"></a>Dodawanie użytkowników do grupy programu SharePoint

Grupa programu SharePoint to zbiór użytkowników, którzy mają ten sam zestaw uprawnień do witryn i zawartości. Zamiast przypisywać uprawnienia jednej osobie, można użyć grup w celu wygodnego przydzielenia tego samego poziomu uprawnień wielu osobom naraz.

- [Dodawanie użytkowników do grupy](https://docs.microsoft.com/sharepoint/customize-sharepoint-site-permissions#add-users-to-a-group)
- [Dodawanie lub usuwanie członków z grup Microsoft 365 przy użyciu Centrum administracyjnego](https://docs.microsoft.com/microsoft-365/admin/create-groups/add-or-remove-members-from-groups)
