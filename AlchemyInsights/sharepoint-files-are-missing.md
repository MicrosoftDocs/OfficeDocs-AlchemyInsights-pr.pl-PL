# <a name="sharepoint-files-are-missing"></a>Brakuje plików programu SharePoint

Jeśli wiele plików usługi OneDrive lub programu SharePoint zostanie usuniętych, zastąpionych, uszkodzonych lub zainfekowanych przez złośliwe oprogramowanie, udostępniamy użytkownikom różne opcje ułatwiające proces odzyskiwania.

Aby uzyskać więcej informacji, zobacz [Jak odzyskać brakujące, usunięte lub uszkodzone elementy w usłudze SharePoint Online i usłudze OneDrive dla firm](https://go.microsoft.com/fwlink/?linkid=2110774).
