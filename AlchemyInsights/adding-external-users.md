# <a name="adding-external-users"></a>Dodawanie użytkowników zewnętrznych

Kroki dodawania użytkowników zewnętrznych/gościnnych mogą się różnić w zależności od tego, czego potrzebują, aby uzyskać do nich dostęp. Wybierz jedną z następujących opcji:
  
- [Omówienie udostępniania zewnętrznego usługi SharePoint Online](https://docs.microsoft.com/sharepoint/external-sharing-overview)

- [Dostęp gościa w grupach usługi Microsoft 365](https://support.office.com/article/guest-access-in-office-365-groups-bfc7a840-868f-4fd6-a390-f347bf51aff6)

- [Dostęp dla gości w usłudze Microsoft Teams](https://docs.microsoft.com/microsoftteams/guest-access-checklist)
