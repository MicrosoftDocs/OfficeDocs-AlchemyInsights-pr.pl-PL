# <a name="got-a-parameter-cannot-be-found-error"></a>Masz błąd "nie można znaleźć parametru"?

Jeśli zostanie wyświetlony komunikat o błędzie **"nie można odnaleźć parametru pasującego do nazwy parametru" EmailAddresses "**, oznacza to, że zajmuje on trochę więcej czasu, w którym program Microsoft 365 kończy konfigurowanie dzierżawy lub domeny niestandardowej, jeśli została wcześniej dodana.
  
Proces konfigurowania może potrwać do 4 godzin. Poczekaj chwilę na ukończenie procesu konfigurowania, a następnie spróbuj ponownie.
