# <a name="outlook-error-cannot-start-microsoft-outlook"></a>Błąd programu Outlook: nie można uruchomić programu Microsoft Outlook

Występują problemy z uruchomieniem programu Outlook 2016, Outlook 2013 lub Outlook 2010 albo otrzymywanie "nie można uruchomić programu Microsoft Outlook. Nie można otworzyć okna programu Outlook "komunikaty o błędach"?
  
Aby rozwiązać te problemy dotyczące uruchamiania, należy najpierw wykonać czynności opisane w [artykule](https://support.office.com/article/I-can-t-start-Microsoft-Outlook-2016-2013-or-2010-or-receive-the-error-Cannot-start-Microsoft-Office-Outlook-Cannot-open-the-Outlook-Window-d1f69da6-b333-4650-97bf-4d77bd7abb85) procedura rozwiązywania problemów. 
  
[Narzędzie Diagnostyka zaawansowana](https://aka.ms/SaRA-OutlookAdvDiagnostics) można również uruchomić na komputerze, którego dotyczy problem.
