# <a name="map-a-sharepoint-library-to-a-network-drive"></a>Mapowanie biblioteki programu SharePoint na dysk sieciowy

Mapowanie biblioteki jako dysku sieciowego jest tymczasowe i obsługiwane tylko przez program Internet Explorer. Zamiast tego [Synchronizuj pliki programu SharePoint z nowym klientem synchronizacji OneDrive](https://support.office.com/article/6de9ede8-5b6e-4503-80b2-6190f3354a88.aspx), który udostępnia [pliki na żądanie](https://support.office.com/article/0e6860d3-d9f3-4971-b321-7092438fb38e.aspx). Dostęp do wszystkich plików w usłudze OneDrive bez użycia lokalnego miejsca do magazynowania.
  
[Uzyskiwanie pomocy przy konfigurowaniu i rozwiązywaniu problemów z mapowanymi dyskami sieciowymi](https://go.microsoft.com/fwlink/?linkid=872946)
