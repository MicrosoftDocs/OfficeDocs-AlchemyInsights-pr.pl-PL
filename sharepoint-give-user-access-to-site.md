# <a name="give-users-access-to-sharepoint-site"></a>Udzielanie użytkownikom dostępu do witryny programu SharePoint

<p>Funkcja żądania dostępu umożliwia osobom na żądanie dostępu do treści, które aktualnie nie mają uprawnień do przeglądania. Będąc właścicielem witryny można skonfigurować tę funkcję, aby wysyłać Ci wiadomości, gdy ktoś żąda dostępu do witryny.
